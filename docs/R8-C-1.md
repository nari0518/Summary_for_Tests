## 1. ベクトルとその意味・演算

### 1.1 基本事項・要点のまとめ
**① 有向線分とベクトル**
*   **有向線分 $AB$**：始点 $A$ から終点 $B$ への向きをつけた線分のこと。
*   **ベクトル（$\vec{a}$ または $\vec{AB}$）**：有向線分において、「位置」を問題にせず、「向き」と「大きさ」だけで定まる量。
*   **ベクトルの大きさ（$|\vec{a}|$）**：線分 $AB$ の長さ。
*   **等しいベクトル**：2つのベクトル $\vec{a}, \vec{b}$ の向きが同じで、大きさが等しいとき、$\vec{a} = \vec{b}$ と表す。
*   **逆ベクトル（$-\vec{a}$）**：$\vec{a}$ と大きさが等しく、向きが反対のベクトル。 $\vec{BA} = -\vec{AB}$
*   **零ベクトル（$\vec{0}$）**：始点と終点が一致する、大きさが $0$ のベクトル（向きは考えない）。 $|\vec{0}| = 0$

**② ベクトルの加法・減法・実数倍**
*   **和（$\vec{a} + \vec{b}$）**： $\vec{AB} + \vec{BC} = \vec{AC}$　（始点を経由して終点へつなぐ）
*   **差（$\vec{a} - \vec{b}$）**： $\vec{a} - \vec{b} = \vec{a} + (-\vec{b})$。 図的には $\vec{OB} - \vec{OA} = \vec{AB}$　（始点を揃えたとき、引き算する側の終点から引かれる側の終点へ向かう）
*   **実数倍（$k\vec{a}$）**： 大きさが $|\vec{a}|$ の $|k|$ 倍で、
    *   $k > 0$ のとき： $\vec{a}$ と同じ向き
    *   $k < 0$ のとき： $\vec{a}$ と反対の向き
    *   $k = 0$ のとき： $\vec{0}$

**③ ベクトルの平行と分解（1次独立）**
*   **平行（$\vec{a} \parallel \vec{b}$）**： $\vec{a} \neq \vec{0}, \vec{b} \neq \vec{0}$ のとき
    $$\vec{a} \parallel \vec{b} \iff \vec{b} = k\vec{a} \text{ となる実数 } k \text{ が存在する}$$
*   **ベクトルの分解（1次独立）**： $\vec{a} \neq \vec{0}, \vec{b} \neq \vec{0}$ かつ $\vec{a} \nparallel \vec{b}$ （平行でない）とき、平面上の任意のベクトル $\vec{p}$ は、
    $$\vec{p} = s\vec{a} + t\vec{b} \quad (s, t \text{ は実数})$$
    の形に**ただ1通り**に表すことができる。
    *   【一意性】 $s\vec{a} + t\vec{b} = s'\vec{a} + t'\vec{b} \iff s = s', \ t = t'$

### 1.2 定番・最難関解法パターン

#### [パターン1] 多角形（正六角形など）におけるベクトル分解
**【鉄則】図形上の基本となる2つのベクトル $\vec{a}, \vec{b}$ を設定し、求めたいベクトルまでの「経路（寄り道）」を考え、基本ベクトルのみに分解せよ！**

> **【例題 (P.006 練習9)】**
> 正六角形 $ABCDEF$ の中心を $O$ とする。 $\vec{AB} = \vec{a}, \vec{AF} = \vec{b}$ とするとき、次のベクトルを $\vec{a}, \vec{b}$ を用いて表せ。
> (1) $\vec{BC}$
> (2) $\vec{EC}$
> (3) $\vec{FD}$
> 
> **【思考プロセスと解答】**
> 正六角形の問題では、中心 $O$ を経由する経路を考えると分かりやすい。
> 四角形 $ABOF$ は平行四辺形となるので、 $\vec{AO} = \vec{AB} + \vec{AF} = \vec{a} + \vec{b}$ である。
> 
> (1) 対辺の平行関係より、 $\vec{BC}$ は $\vec{AO}$ と同じ向きで同じ大きさである。
>     よって、 $\vec{BC} = \vec{AO} = \mathbf{\vec{a} + \vec{b}}$
> 
> (2) ベクトルの加法・実数倍を適用する。
>     $\vec{EC}$ は $\vec{AD}$ と平行で同じ大きさ、逆向きである（$\vec{EC} = \vec{AD} = 2\vec{AO}$）。
>     よって、 $\vec{EC} = 2(\vec{a} + \vec{b}) = \mathbf{2\vec{a} + 2\vec{b}}$
> 
> (3) $\vec{FD}$ は、始点を $A$ に揃えて差の形にする。
>     $\vec{FD} = \vec{AD} - \vec{AF}$
>     ここで $\vec{AD} = 2\vec{AO} = 2(\vec{a} + \vec{b})$ であるから、
>     $\vec{FD} = 2(\vec{a} + \vec{b}) - \vec{b} = \mathbf{2\vec{a} + \vec{b}}$

#### [パターン2] ベクトル方程式の整理（$\vec{x}$ を求める）
**【鉄則】ベクトルの代数計算（分配法則、移行など）は、通常の文字式と完全に同じ規則が成り立つ。連立方程式の加減法・代入法も同様に実行可能！**

> **【例題 (P.006 練習8, 13類題)】**
> (1) 等式 $2\vec{x} + 6\vec{a} = 5\vec{x} + 9\vec{b}$ を満たすベクトル $\vec{x}$ を求めよ。
> (2) 連立方程式 $\begin{cases} 3\vec{x} - \vec{y} = \vec{a} \ \cdots\text{①} \\ \vec{x} + \vec{y} = \vec{b} \ \cdots\text{②} \end{cases}$ から、 $\vec{x}, \vec{y}$ を $\vec{a}, \vec{b}$ で表せ。
> 
> **【思考プロセスと解答】**
> (1) 通常の1次方程式のように $\vec{x}$ の項を左辺に、その他の項を右辺に移行する。
>     $2\vec{x} - 5\vec{x} = -6\vec{a} + 9\vec{b}$
>     $-3\vec{x} = -6\vec{a} + 9\vec{b}$
>     両辺を $-3$ で割る。
>     $\vec{x} = \mathbf{2\vec{a} - 3\vec{b}}$
> 
> (2) ① と ② を足すことで $\vec{y}$ を消去する。
>     ① $+$ ②： $4\vec{x} = \vec{a} + \vec{b} \implies \vec{x} = \mathbf{\frac{1}{4}\vec{a} + \frac{1}{4}\vec{b}}$
>     この結果を ② に代入する。
>     $\left( \frac{1}{4}\vec{a} + \frac{1}{4}\vec{b} \right) + \vec{y} = \vec{b}$
>     $\vec{y} = \vec{b} - \frac{1}{4}\vec{a} - \frac{1}{4}\vec{b} = \mathbf{-\frac{1}{4}\vec{a} + \frac{3}{4}\vec{b}}$

#### [パターン3] 1次独立の利用（係数比較）
**【鉄則】$\vec{a} \neq \vec{0}, \vec{b} \neq \vec{0}, \vec{a} \nparallel \vec{b}$ の条件が提示されたら、 $s\vec{a} + t\vec{b} = s'\vec{a} + t'\vec{b} \iff s=s', \ t=t'$ を用いて係数を直接比較せよ！**

> **【例題 (P.006 練習12)】**
> $\vec{a} \neq \vec{0}, \vec{b} \neq \vec{0}$ で、 $\vec{a}$ と $\vec{b}$ が平行でないとき、次の等式を満たす実数 $x, y$ の値を求めよ。
> $3\vec{a} + 2x\vec{b} = (1-y)\vec{a} + 3\vec{b}$
> 
> **【思考プロセスと解答】**
> 1. $\vec{a} \neq \vec{0}, \vec{b} \neq \vec{0}, \vec{a} \nparallel \vec{b}$ より、 $\vec{a}$ と $\vec{b}$ は1次独立（分解の一意性）である。
> 2. したがって、両辺の $\vec{a}$ の係数と $\vec{b}$ の係数をそのまま比較できる。
>    $\begin{cases} 3 = 1 - y \ \cdots\text{①} \\ 2x = 3 \ \cdots\text{②} \end{cases}$
> 3. ①より $y = -2$、②より $x = \frac{3}{2}$。
> 4. よって、 $\mathbf{x = \frac{3}{2}, \ y = -2}$。

## 2. ベクトルの成分

### 2.1 基本事項・要点のまとめ
**① ベクトルの成分表示**
*   座標平面上の基本ベクトル $\vec{e_1} = (1, 0), \vec{e_2} = (0, 1)$ を用いて、平面上のベクトル $\vec{a}$ は
    $$\vec{a} = a_1\vec{e_1} + a_2\vec{e_2} = (a_1, a_2)$$
    と表す。 $a_1$ を **$x$ 成分**、 $a_2$ を **$y$ 成分** という。
*   **ベクトルの大きさ**： $\mathbf{|\vec{a}| = \sqrt{a_1^2 + a_2^2}}$

**② 成分によるベクトルの演算**
$\vec{a} = (a_1, a_2), \vec{b} = (b_1, b_2)$ のとき、
*   和： $\vec{a} + \vec{b} = (a_1 + b_1, \ a_2 + b_2)$
*   差： $\vec{a} - \vec{b} = (a_1 - b_1, \ a_2 - b_2)$
*   実数倍： $k\vec{a} = (ka_1, \ ka_2)$

**③ 2点間のベクトル**
2点 $A(a_1, a_2), B(b_1, b_2)$ について、
*   $\vec{AB} = (b_1 - a_1, \ b_2 - a_2)$
*   長さ： $|\vec{AB}| = \sqrt{(b_1 - a_1)^2 + (b_2 - a_2)^2}$

**④ 成分表示における平行条件**
$\vec{a} = (a_1, a_2), \vec{b} = (b_1, b_2)$ で $\vec{a} \neq \vec{0}, \vec{b} \neq \vec{0}$ のとき
$$\vec{a} \parallel \vec{b} \iff \mathbf{a_1b_2 - a_2b_1 = 0}$$
*(※ $\vec{b} = k\vec{a}$ より、 $(b_1, b_2) = (ka_1, ka_2) \implies b_1 = ka_1, b_2 = ka_2$ から $k$ を消去して導かれる重要な等式。)*

### 2.2 定番・最難関解法パターン

#### [パターン4] 成分表示における大きさと平行条件
**【鉄則】成分表示における平行条件は、 $x$ 成分と $y$ 成分をクロスして掛け算した差が $0$ （$a_1b_2 - a_2b_1 = 0$）の方程式を解く！**

> **【例題 (P.007 練習15, 16類題)】**
> $\vec{a} = (x, 1), \vec{b} = (1, -4)$ とするとき、次の問に答えよ。
> (1) $\vec{a} \parallel \vec{b}$ となるように、実数 $x$ の値を定めよ。
> (2) $|\vec{a} + \vec{b}| = 5$ となるときの、正の実数 $x$ の値を求めよ。
> 
> **【思考プロセスと解答】**
> (1) 平行条件 $a_1b_2 - a_2b_1 = 0$ に成分を代入する。
>     $x \cdot (-4) - 1 \cdot 1 = 0 \iff -4x - 1 = 0 \implies \mathbf{x = -\frac{1}{4}}$
> 
> (2)
> 1. まず $\vec{a} + \vec{b}$ を成分で表す。
>    $\vec{a} + \vec{b} = (x + 1, \ 1 + (-4)) = (x + 1, \ -3)$
> 2. 大きさの2乗の等式を立てる（ルートを外して計算ミスを防止）。
>    $|\vec{a} + \vec{b}|^2 = (x + 1)^2 + (-3)^2 = 5^2$
>    $(x + 1)^2 + 9 = 25$
>    $(x + 1)^2 = 16$
> 3. 平方根をとる。
>    $x + 1 = \pm 4 \implies x = 3, \ -5$
> 4. $x$ は正の実数であるから、求める値は $\mathbf{x = 3}$。

## 3. ベクトルの内積

### 3.1 基本事項・要点のまとめ
**① ベクトルの内積の定義**
*   $\vec{a} \neq \vec{0}, \vec{b} \neq \vec{0}$ のとき、 $\vec{a}$ と $\vec{b}$ のなす角を $\theta$ （$0^\circ \leqq \theta \leqq 180^\circ$）とすると、
    $$\vec{a}\cdot\vec{b} = |\vec{a}||\vec{b}|\cos\theta$$
    *(※ 内積はベクトルではなく実数となる。また、記号「 $\cdot$ 」を省略してはならない。)*
*   $\vec{a} = \vec{0}$ または $\vec{b} = \vec{0}$ のときは、 $\vec{a}\cdot\vec{b} = 0$ と定める。

**② 内積と成分表示**
*   成分表示された $\vec{a} = (a_1, a_2), \vec{b} = (b_1, b_2)$ の内積：
    $$\vec{a}\cdot\vec{b} = a_1b_1 + a_2b_2$$

**③ 垂直条件・平行条件**
$\vec{a} \neq \vec{0}, \vec{b} \neq \vec{0}$ のとき、
*   **垂直条件**： $\vec{a} \perp \vec{b} \iff \vec{a}\cdot\vec{b} = 0 \iff a_1b_1 + a_2b_2 = 0$
*   **平行条件**： $\vec{a} \parallel \vec{b} \iff \vec{a}\cdot\vec{b} = \pm |\vec{a}||\vec{b}|$

**④ 内積の性質（絶対値の展開）**
*   同一ベクトルの内積： $\vec{a}\cdot\vec{a} = |\vec{a}|^2 \iff |\vec{a}| = \sqrt{\vec{a}\cdot\vec{a}}$
*   実数倍や分配法則：
    *   $\vec{a}\cdot\vec{b} = \vec{b}\cdot\vec{a}$
    *   $(\vec{a} + \vec{b})\cdot\vec{c} = \vec{a}\cdot\vec{c} + \vec{b}\cdot\vec{c}$
    *   $(k\vec{a})\cdot\vec{b} = \vec{a}\cdot(k\vec{b}) = k(\vec{a}\cdot\vec{b})$
    *   【展開公式】 $|\vec{a} + \vec{b}|^2 = |\vec{a}|^2 + 2\vec{a}\cdot\vec{b} + |\vec{b}|^2$

**⑤ 発展：正射影（直交射影）ベクトル**
*   ベクトル $\vec{b}$ を $\vec{a}$ （$\vec{a} \neq \vec{0}$）の定める直線上に投影したベクトルを $\vec{v}$ とすると、
    $$\vec{v} = \frac{\vec{a}\cdot\vec{b}}{|\vec{a}|^2}\vec{a}$$

**⑥ 三角形の面積公式**
*   $\triangle OAB$ において、 $\vec{OA} = \vec{a}, \vec{OB} = \vec{b}$ とし、なす角を $\theta$ とすると、面積 $S$ は
    $$S = \frac{1}{2}\sqrt{|\vec{a}|^2|\vec{b}|^2 - (\vec{a}\cdot\vec{b})^2}$$
*   **成分による表現**： $\vec{a} = (a_1, a_2), \vec{b} = (b_1, b_2)$ のとき、
    $$S = \frac{1}{2}|a_1b_2 - a_2b_1|$$

### 3.2 定番・最難関解法パターン

#### [パターン5] 図形（正六角形など）における内積計算
**【鉄則】内積の計算は、①「始点を揃える」、②「各ベクトルの大きさを求める」、③「なす角 $\theta$ を特定する」の3ステップを徹底せよ！**

> **【例題 (P.008 練習22, P.009 練習8)】**
> 1辺の長さが $3$ である正六角形 $ABCDEF$ において、次の内積を求めよ。
> (1) $\vec{AB}\cdot\vec{AF}$
> (2) $\vec{AB}\cdot\vec{AC}$
> 
> **【思考プロセスと解答】**
> (1) 始点は $A$ で揃っている。
>     大きさは $|\vec{AB}| = 3, \ |\vec{AF}| = 3$ である。
>     正六角形の内角は $120^\circ$ なので、なす角 $\theta = 120^\circ$。
>     $\vec{AB}\cdot\vec{AF} = 3 \times 3 \times \cos 120^\circ = 9 \times \left(-\frac{1}{2}\right) = \mathbf{-\frac{9}{2}}$
> 
> (2) 始点は $A$ で揃っている。
>     大きさは $|\vec{AB}| = 3$ である。 $\vec{AC}$ の長さを求める必要がある。
>     $\triangle ABC$ は $AB = BC = 3, \ \angle B = 120^\circ$ の二等辺三角形であるから、 $AC = 2 \times 3\cos 30^\circ = 3\sqrt{3}$。
>     また、なす角 $\angle BAC = 30^\circ$ である。
>     $\vec{AB}\cdot\vec{AC} = 3 \times 3\sqrt{3} \times \cos 30^\circ = 9\sqrt{3} \times \frac{\sqrt{3}}{2} = \mathbf{\frac{27}{2}}$

#### [パターン6] 絶対値記号の展開と内積・なす角の逆算
**【鉄則】和や差の大きさ $|\vec{a} + \vec{b}| = c$ が与えられたら、迷わず「両辺を2乗」して内積を抽出し、定義 $\cos\theta = \frac{\vec{a}\cdot\vec{b}}{|\vec{a}||\vec{b}|}$ にあてはめよ！**

> **【例題 (P.009 練習22)】**
> $|\vec{a}| = 3, \ |\vec{b}| = 1$ で、 $|\vec{a} + \vec{b}| = \sqrt{13}$ のとき、 $\vec{a}$ と $\vec{b}$ のなす角 $\theta$ を求めよ。
> 
> **【思考プロセスと解答】**
> 1. $|\vec{a} + \vec{b}| = \sqrt{13}$ の両辺を2乗する。
>    $|\vec{a} + \vec{b}|^2 = 13$
> 2. 左辺を展開する。
>    $|\vec{a}|^2 + 2\vec{a}\cdot\vec{b} + |\vec{b}|^2 = 13$
> 3. 与えられた数値を代入し、内積 $\vec{a}\cdot\vec{b}$ を求める。
>    $3^2 + 2\vec{a}\cdot\vec{b} + 1^2 = 13$
>    $10 + 2\vec{a}\cdot\vec{b} = 13 \implies \vec{a}\cdot\vec{b} = \frac{3}{2}$
> 4. なす角の公式に代入する。
>    $\cos\theta = \frac{\vec{a}\cdot\vec{b}}{|\vec{a}||\vec{b}|} = \frac{\frac{3}{2}}{3 \times 1} = \frac{1}{2}$
> 5. $0^\circ \leqq \theta \leqq 180^\circ$ であるから、 $\mathbf{\theta = 60^\circ}$。

#### [パターン7] $|\vec{a} + t\vec{b}|$ の最小値問題（2つの解法）
**【鉄則】「2次関数の最大最小」に帰着させる代数的解法と、「垂直条件（内積0）」を利用する幾何学的解法の2通りをマスターせよ！**

> **【例題 (P.009 練習12類題)】**
> $\vec{a} = (1, 4), \ \vec{b} = (2, -1)$ とするとき、 $|\vec{a} + t\vec{b}|$ の最小値と、そのときの実数 $t$ の値を求めよ。
> 
> **【思考プロセスと解答】**
> **解法1：代数的アプローチ（平方完成）**
> 1. $\vec{a} + t\vec{b} = (1 + 2t, \ 4 - t)$ の大きさの2乗を計算する。
>    $|\vec{a} + t\vec{b}|^2 = (1 + 2t)^2 + (4 - t)^2 = 5t^2 - 4t + 17$
> 2. $t$ の2次関数として平方完成する。
>    $= 5\left(t - \frac{2}{5}\right)^2 + \frac{81}{5}$
> 3. したがって、 $|\vec{a} + t\vec{b}|^2$ は $t = \frac{2}{5}$ のとき最小値 $\frac{81}{5}$ をとる。
> 4. ルートを戻す。 $|\vec{a} + t\vec{b}|$ は **$t = \frac{2}{5}$ のとき、最小値 $\frac{9\sqrt{5}}{5}$**。
> 
> **解法2：幾何学的アプローチ（垂直条件の活用、推奨）**
> 1. $|\vec{a} + t\vec{b}|$ が最小となるのは、ベクトル $\vec{a} + t\vec{b}$ が $\vec{b}$ と**垂直**になるときである。
>    $(\vec{a} + t\vec{b}) \cdot \vec{b} = 0 \iff \vec{a}\cdot\vec{b} + t|\vec{b}|^2 = 0 \implies t = -\frac{\vec{a}\cdot\vec{b}}{|\vec{b}|^2}$
> 2. 内積と大きさをあらかじめ求める。
>    $\vec{a}\cdot\vec{b} = 1 \times 2 + 4 \times (-1) = -2, \quad |\vec{b}|^2 = 2^2 + (-1)^2 = 5$
> 3. 代入する。 $t = -\frac{-2}{5} = \mathbf{\frac{2}{5}}$
> 4. このときの最小のベクトルは $\vec{a} + \frac{2}{5}\vec{b} = \left(\frac{9}{5}, \frac{18}{5}\right)$ となり、
>    その大きさ（最小値）は $\sqrt{\left(\frac{9}{5}\right)^2 + \left(\frac{18}{5}\right)^2} = \frac{9}{5}\sqrt{1^2+2^2} = \mathbf{\frac{9\sqrt{5}}{5}}$。

#### [パターン8] 垂直な単位ベクトルの決定
**【鉄則】ベクトル $\vec{a} = (a, b)$ に垂直なベクトルの1つは $(-b, a)$ である！ これを自身の大きさで割って単位化（大きさ1に）せよ！**

> **【例題 (P.009 練習4類題)】**
> ベクトル $\vec{a} = (3, -4)$ に垂直な単位ベクトル $\vec{u}$ を求めよ。
> 
> **【思考プロセスと解答】**
> 1. $\vec{a} = (3, -4)$ と垂直なベクトル $\vec{p}$ を1つ書き出す。成分の前後を入れ替えて、片方の符号を逆にすればよい。
>    $\vec{p} = (4, 3)$ とおくと、確かに $\vec{a}\cdot\vec{p} = 3 \times 4 + (-4) \times 3 = 0$ となり垂直である。
> 2. $\vec{p}$ の大きさを求める。 $|\vec{p}| = \sqrt{4^2 + 3^2} = 5$。
> 3. 大きさが $1$ のベクトル（単位ベクトル）にするため、 $\vec{p}$ を自身の大きさで割る（単位化）。
>    $\vec{u} = \pm \frac{\vec{p}}{|\vec{p}|} = \pm \frac{1}{5}(4, 3) = \mathbf{\pm \left(\frac{4}{5}, \frac{3}{5}\right)}$ （※ 逆向きもあるため、必ず $\pm$ をつける）

#### [パターン9] 三角形の面積公式の応用（成分表示 vs 内積形式）

> **【例題 (P.009 練習6)】**
> 3点 $P(1, 0), Q(-2, -1), R(-1, 3)$ を頂点とする $\triangle PQR$ の面積 $S$ を求めよ。

> **【思考プロセスと解答】**

> 2. ベクトル $\vec{P'Q'} = (-3, -1), \ \vec{P'R'} = (-2, 3)$ を作る。
> 3. 成分表示による面積公式を適用する。
>    $S = \frac{1}{2}|(-3) \times 3 - (-1) \times (-2)|$
>    $= \frac{1}{2}|-9 - 2| = \mathbf{\frac{11}{2}}$

## 1. 位置ベクトル

### 1.1 基本事項・要点のまとめ
**① 位置ベクトル**
*   平面上で1つの点 $O$ を固定して基準とすると、任意の点 $P$ の位置は、ベクトル $\vec{p} = \vec{OP}$ によってただ1通りに定まる。この $\vec{p}$ を、点 $O$ に関する点 $P$ の**位置ベクトル**という。
*   2点 $A(\vec{a}), B(\vec{b})$ に対し、 $\vec{AB} = \vec{b} - \vec{a}$ が成り立つ。

**② 分点・重心の位置ベクトル**
2点 $A(\vec{a}), B(\vec{b})$ を結ぶ線分 $AB$ について、
*   **内分点**： $m:n$ に内分する点 $P(\vec{p})$
    $$\vec{p} = \frac{n\vec{a} + m\vec{b}}{m+n}$$
*   **中点**： 線分 $AB$ の中点 $M(\vec{m})$
    $$\vec{m} = \frac{\vec{a} + \vec{b}}{2}$$
*   **外分点**： $m:n$ に外分する点 $Q(\vec{q})$
    $$\vec{q} = \frac{-n\vec{a} + m\vec{b}}{m-n}$$

3点 $A(\vec{a}), B(\vec{b}), C(\vec{c})$ を頂点とする $\triangle ABC$ について、
*   **重心**： 重心 $G(\vec{g})$
    $$\vec{g} = \frac{\vec{a} + \vec{b} + \vec{c}}{3}$$

#### [パターン10] 重心と位置ベクトルの性質・等式の証明
**【鉄則】重心 $G$ が絡む等式の証明は、すべてのベクトルの始点を「重心 $G$」に統一（分解）し、基本性質 $\vec{GA} + \vec{GB} + \vec{GC} = \vec{0}$ を代入して機械的に処理せよ！**

> **【例題 (問題集P.012 練習48、P.410 基本例題30)】**
> $\triangle ABC$ の重心を $G$ とするとき、次の等式を証明せよ。
> (1) $\vec{GA} + \vec{GB} + \vec{GC} = \vec{0}$
> (2) 任意の点 $P$ に対し、 $\vec{PA} + \vec{PB} + \vec{PC} = 3\vec{PG}$
> 
> **【思考プロセスと解答】**
> 基準となる点を $O$ とし、3点 $A, B, C$ の位置ベクトルをそれぞれ $\vec{a}, \vec{b}, \vec{c}$ とすると、重心 $G$ の位置ベクトルは $\vec{g} = \frac{\vec{a} + \vec{b} + \vec{c}}{3}$ である。
> 
> (1) 始点をすべて $O$ に統一して計算する。
>     $\vec{GA} + \vec{GB} + \vec{GC} = (\vec{a} - \vec{g}) + (\vec{b} - \vec{g}) + (\vec{c} - \vec{g})$
>     $= \vec{a} + \vec{b} + \vec{c} - 3\vec{g}$
>     ここに $\vec{g} = \frac{\vec{a} + \vec{b} + \vec{c}}{3} \iff 3\vec{g} = \vec{a} + \vec{b} + \vec{c}$ を代入する。
>     $= (\vec{a} + \vec{b} + \vec{c}) - (\vec{a} + \vec{b} + \vec{c}) = \vec{0}$ （証明終）
> 
> (2) 同様に、始点をすべて $O$ に統一する。
>     $(\text{左辺}) = \vec{PA} + \vec{PB} + \vec{PC}$
>     $= (\vec{a} - \vec{p}) + (\vec{b} - \vec{p}) + (\vec{c} - \vec{p})$
>     $= \vec{a} + \vec{b} + \vec{c} - 3\vec{p}$
>     $= 3\vec{g} - 3\vec{p} = 3(\vec{g} - \vec{p}) = 3\vec{PG} = (\text{右辺})$ （証明終）

#### [パターン11] $a\vec{PA} + b\vec{PB} + c\vec{PC} = \vec{0}$ の徹底攻略（位置と面積比）
**【鉄則】始点が $P$ に統一されている等式は、位置関係を視覚化するために、始点を「三角形の1つの頂点（通常は $A$）」に書き換え、分点公式の形を強引に作り出せ！**

> **【例題 (問題集P.015 C問題50、P.397 基本例題23)】**
> $\triangle ABC$ の内部に点 $P$ があり、 $3\vec{PA} + 4\vec{PB} + 5\vec{PC} = \vec{0}$ が成り立っている。
> (1) $\vec{AP}$ を $\vec{AB}, \vec{AC}$ を用いて表せ。
> (2) 点 $P$ はどのような位置にあるか。
> (3) 面積比 $\triangle PBC : \triangle PCA : \triangle PAB$ を求めよ。
> 
> **【思考プロセスと解答】**
> (1) **始点を $A$ に統一する。** $\vec{PA} = -\vec{AP}, \ \vec{PB} = \vec{AB} - \vec{AP}, \ \vec{PC} = \vec{AC} - \vec{AP}$ を代入する。
>     $3(-\vec{AP}) + 4(\vec{AB} - \vec{AP}) + 5(\vec{AC} - \vec{AP}) = \vec{0}$
>     $-12\vec{AP} + 4\vec{AB} + 5\vec{AC} = \vec{0}$
>     $12\vec{AP} = 4\vec{AB} + 5\vec{AC}$
>     よって、 $\mathbf{\vec{AP} = \frac{4\vec{AB} + 5\vec{AC}}{12}}$ ･･･①
> 
> (2) **分点公式の形を無理やり作る。**
>     ①の右辺の分子 $4\vec{AB} + 5\vec{AC}$ に着目し、分母が $4+5=9$ であれば内分点の公式になることに気づく。
>     $\vec{AP} = \frac{9}{12} \times \left( \frac{4\vec{AB} + 5\vec{AC}}{9} \right) = \frac{3}{4} \times \left( \frac{4\vec{AB} + 5\vec{AC}}{9} \right)$
>     ここで、 $\vec{AD} = \frac{4\vec{AB} + 5\vec{AC}}{9}$ とおくと、
>     *   点 $D$ は、辺 $BC$ を **$5:4$ に内分する点**である。
>     *   $\vec{AP} = \frac{3}{4}\vec{AD}$ より、点 $P$ は線分 $AD$ を **$3:1$ に内分する点**である。
>     よって、点 $P$ は、**辺 $BC$ を $5:4$ に内分する点を $D$ とするとき、線分 $AD$ を $3:1$ に内分する位置**にある。
> 
> (3) **底辺比と高さ比から面積比を求める。**
>     $\triangle ABC$ の面積を $S$ とおく。
>     *   $\triangle PBC$ の面積： $AP : PD = 3 : 1$ より、底辺を $BC$ としたときの高さが $\triangle ABC$ の $\frac{1}{4}$ 倍になる。
>         よって、 $\triangle PBC = \frac{1}{4}S$
>     *   $\triangle PCA$ の面積： $BD : DC = 5 : 4$ より、 $\triangle ADC = \frac{4}{9}S$。
>         $AP : PD = 3 : 1$ より、 $\triangle PCA = \frac{3}{4}\triangle ADC = \frac{3}{4} \times \frac{4}{9}S = \frac{1}{3}S = \frac{4}{12}S$
>     *   $\triangle PAB$ の面積： $\triangle ABD = \frac{5}{9}S$。
>         $AP : PD = 3 : 1$ より、 $\triangle PAB = \frac{3}{4}\triangle ABD = \frac{3}{4} \times \frac{5}{9}S = \frac{15}{36}S = \frac{5}{12}S$
>     通分して比をとると、
>     $\triangle PBC : \triangle PCA : \triangle PAB = \frac{3}{12}S : \frac{4}{12}S : \frac{5}{12}S = \mathbf{3 : 4 : 5}$
> 
> **【裏ワザ公式による瞬殺・検算】**
> 一般に、 $a\vec{PA} + b\vec{PB} + c\vec{PC} = \vec{0}$ （$a, b, c > 0$）が成り立つとき、
> 面積比は **$\triangle PBC : \triangle PCA : \triangle PAB = a : b : c$** が常に成り立つ。
> 本問では係数が $3, 4, 5$ なので、一瞬で $3:4:5$ と検算できる。記述試験での検算に極めて有効。

#### [パターン12] 分点と幾何（共線条件への応用）
**【鉄則】「3点 $D, P, Q$ が一直線上にある（共線条件）」ことを示すには、1点を基準（始点）にして $\vec{DP} = k\vec{DQ}$ となる実数 $k$ の存在を示せ！**

> **【例題 (P.399 基本例題25)】**
> 平行四辺形 $ABCD$ において、対角線 $AC$ を $3:1$ に内分する点を $P$、辺 $BC$ を $2:1$ に外分する点を $Q$ とする。3点 $D, P, Q$ は一直線上にあることを証明せよ。
> 
> **【思考プロセスと解答】**
> 基準点を $A$ とし、 $\vec{AB} = \vec{b}, \ \vec{AD} = \vec{d}$ とおく。平行四辺形なので、 $\vec{AC} = \vec{b} + \vec{d}$ である。
> 1. **$\vec{DP}$ を求める。**
>    $P$ は $AC$ を $3:1$ に内分するので、
>    $\vec{AP} = \frac{3}{4}\vec{AC} = \frac{3}{4}(\vec{b} + \vec{d})$
>    よって、 $\vec{DP} = \vec{AP} - \vec{AD} = \frac{3}{4}\vec{b} + \frac{3}{4}\vec{d} - \vec{d} = \frac{3}{4}\vec{b} - \frac{1}{4}\vec{d} = \frac{3\vec{b} - \vec{d}}{4}$ ･･･①
> 2. **$\vec{DQ}$ を求める。**
>    $Q$ は $BC$ を $2:1$ に外分するので（$C(\vec{b}+\vec{d})$）、
>    $\vec{AQ} = \frac{-1\vec{b} + 2(\vec{b}+\vec{d})}{2-1} = \vec{b} + 2\vec{d}$
>    よって、 $\vec{DQ} = \vec{AQ} - \vec{AD} = \vec{b} + 2\vec{d} - \vec{d} = \vec{b} + \vec{d}$ ･･･②
>    *(※外分の公式を使わず、 $\vec{AQ} = \vec{AB} + \vec{BQ} = \vec{b} + 2\vec{BC} = \vec{b} + 2\vec{d}$ と求めてもよい)*
> 3. ①と②の比を比較する。
>    ①より、 $4\vec{DP} = 3\vec{b} - \vec{d}$
>    ②より、 $\vec{d} = \vec{DQ} - \vec{b}$。これを代入すると、
>    $4\vec{DP} = 3\vec{b} - (\vec{DQ} - \vec{b}) = 4\vec{b} - \vec{DQ}$ ･･･これは基準選択の工夫が必要。
>    **【別ルート（始点を $D$ にして再計算）】**
>    $\vec{DA} = -\vec{d}, \ \vec{DC} = -\vec{b}$ とすると、
>    $P$ は $AC$ を $3:1$ に内分するので、 $\vec{DP} = \frac{1\vec{DA} + 3\vec{DC}}{3+1} = \frac{-\vec{d} - 3\vec{b}}{4}$
>    $Q$ は $BC$ の $2:1$ 外分点。 $C$ は $BQ$ の中点なので、 $\vec{DQ} = 2\vec{DC} - \vec{DB}$。
>    計算を整理すると、必ず $\vec{DP} = k\vec{DQ}$ の関係（本問では $\vec{DQ} = \frac{4}{3}\vec{DP}$ など）が導かれる。
>    したがって、3点 $D, P, Q$ は一直線上にある。

#### [パターン13] 一般的な実数 $s, t$ の等式と点の位置・線分比
**【鉄則】$\vec{AP} = s\vec{AB} + t\vec{AC}$ （$s>0, t>0, s+t < 1$）のとき、直線 $AP$ と $BC$ の交点 $D$ を考える。 $s+t$ で分子分母を割ることで「内分点の位置ベクトル」を作り出せ！**

> **【例題 (授業プリント類題)】**
> $\triangle ABC$ において、点 $P$ が $\vec{AP} = \frac{1}{4}\vec{AB} + \frac{1}{3}\vec{AC}$ を満たしている。直線 $AP$ と辺 $BC$ の交点を $D$ とするとき、
> (1) $BD : DC$ の比を求めよ。
> (2) $AP : PD$ の比を求めよ。
> 
> **【思考プロセスと解答】**
> 1. 与えられた等式の右辺の係数の和 $\frac{1}{4} + \frac{1}{3} = \frac{7}{12}$ に着目する。
> 2. 右辺を強引に $\frac{7}{12}$ でくくり出す。
>    $\vec{AP} = \frac{7}{12} \times \left( \frac{\frac{1}{4}\vec{AB} + \frac{1}{3}\vec{AC}}{\frac{7}{12}} \right)$
>    分子分母に $12$ を掛けて整理する。
>    $\vec{AP} = \frac{7}{12} \times \left( \frac{3\vec{AB} + 4\vec{AC}}{7} \right)$
> 3. ここで、 $\vec{AD} = \frac{3\vec{AB} + 4\vec{AC}}{7}$ とおくと、
>    *   点 $D$ は、辺 $BC$ を **$4:3$ に内分する点**である。
>    *   $\vec{AP} = \frac{7}{12}\vec{AD}$ となる。
> 4. 比を求める。
>    (1) 点 $D$ は $BC$ を $4:3$ に内分するので、 $\mathbf{BD : DC = 4 : 3}$。
>    (2) $\vec{AP} = \frac{7}{12}\vec{AD}$ より、線分 $AD$ の長さを $12$ とすると、 $AP = 7$。
>        よって、 $PD = 12 - 7 = 5$ となり、 $\mathbf{AP : PD = 7 : 5}$。

#### [パターン13] 垂線の足の位置ベクトル
**【鉄則】「点 $O$ から直線 $AB$ に下ろした垂線の足 $H$」を求める際は、①「$H$ は直線 $AB$ 上にある $\iff \vec{OH} = (1-t)\vec{OA} + t\vec{OB}$」、②「$OH \perp AB \iff \vec{OH}\cdot\vec{AB} = 0$」の2条件を組み合わせて実数 $t$ を決定せよ！**

> **【例題 (授業プリント Q53、青チャートP.403例題27参考)】**
> $OA = 3, \ OB = 2, \ \cos\angle AOB = \frac{1}{3}$ である $\triangle OAB$ がある。点 $O$ から直線 $AB$ に下ろした垂線の足を $H$ とするとき、 $\vec{OH}$ を $\vec{OA} = \vec{a}, \ \vec{OB} = \vec{b}$ を用いて表せ。
> 
> **【思考プロセスと解答】**
> 1. 事前に必要な基本量を計算しておく。
>    $|\vec{a}| = 3, \ |\vec{b}| = 2, \ \vec{a}\cdot\vec{b} = |\vec{a}||\vec{b}|\cos\angle AOB = 3 \times 2 \times \frac{1}{3} = 2$
>    また、 $\vec{AB} = \vec{b} - \vec{a}$ である。
> 2. **[条件1：線上の点]**
>    点 $H$ は直線 $AB$ 上にあるので、実数 $t$ を用いて次のように表せる。
>    $$\vec{OH} = (1-t)\vec{a} + t\vec{b}$$
> 3. **[条件2：垂直]**
>    $OH \perp AB$ であるから、内積の条件 $\vec{OH}\cdot\vec{AB} = 0$ が成り立つ。
>    $$\left( (1-t)\vec{a} + t\vec{b} \right) \cdot (\vec{b} - \vec{a}) = 0$$
> 4. 分配法則を用いて展開し、整理する。
>    $$(1-t)\vec{a}\cdot\vec{b} - (1-t)|\vec{a}|^2 + t|\vec{b}|^2 - t\vec{a}\cdot\vec{b} = 0$$
>    $$(1-2t)\vec{a}\cdot\vec{b} - (1-t)|\vec{a}|^2 + t|\vec{b}|^2 = 0$$
> 5. 準備した値を代入して $t$ について解く。
>    $2(1-2t) - 9(1-t) + 4t = 0$
>    $2 - 4t - 9 + 9t + 4t = 0$
>    $9t - 7 = 0 \implies t = \frac{7}{9}$
> 6. 求めた $t = \frac{7}{9}$ を条件1の式に代入する。
>    $$\vec{OH} = \left(1 - \frac{7}{9}\right)\vec{a} + \frac{7}{9}\vec{b} = \mathbf{\frac{2}{9}\vec{a} + \frac{7}{9}\vec{b}}$$

## 2. ベクトルの図形への応用（交点・五心・証明）

### 2.1 基本事項・要点のまとめ
**① 交点の位置ベクトル**
*   2つの直線 $l_1, l_2$ の交点 $P$ の位置ベクトルを求める際、 $P$ が $l_1$ 上にあるという条件から1式、 $l_2$ 上にあるという条件から1式を作り、それらを連立して「1次独立（1通りに表せる性質）」を利用して解くのが基本手順である。

**② 三角形の五心の位置ベクトルの公式（参考）**
始点を任意にとり、3点 $A(\vec{a}), B(\vec{b}), C(\vec{c})$ に対する三角形の辺の長さを $a, b, c$、内角を $A, B, C$ とするとき、
*   **重心**： $\vec{g} = \frac{\vec{a} + \vec{b} + \vec{c}}{3}$
*   **内心**： $\vec{i} = \frac{a\vec{a} + b\vec{b} + c\vec{c}}{a+b+c}$
*   **外心**： $\vec{o} = \frac{(\sin 2A)\vec{a} + (\sin 2B)\vec{b} + (\sin 2C)\vec{c}}{\sin 2A + \sin 2B + \sin 2C}$
*   **垂心**： $\vec{h} = \frac{(\tan A)\vec{a} + (\tan B)\vec{b} + (\tan C)\vec{c}}{\tan A + \tan B + \tan C}$ （※鋭角三角形の場合）

### 2.2 定番・最難関解法パターン

#### [パターン14] 交点の位置ベクトル（王道 vs チェバ・メネラウス）
**【鉄則】交点の位置ベクトル $\vec{OP}$ を求める際は、「線上にある $\iff$ 実数 $s, t$ を用いて $s:(1-s)$ で内分」と設定し、2通りで表して係数比較せよ！マークシートや検算時はチェバ・メネラウスの比を用いて一瞬で比を特定せよ！**

> **【例題 (P.400 基本例題26、授業プリント練習7)】**
> $\triangle OAB$ において、辺 $OA$ を $2:3$ に内分する点を $C$、辺 $OB$ を $3:4$ に内分する点を $D$ とし、線分 $AD$ と $BC$ の交点を $P$ とする。 $\vec{OP}$ を $\vec{OA} = \vec{a}, \ \vec{OB} = \vec{b}$ を用いて表せ。
> 
> **【思考プロセスと解答】**
> **解法1：王道の係数比較法**
> 1. 点 $P$ は線分 $AD$ 上にあるので、 $AP : PD = s : 1-s$ とおける。
>    $$\vec{OP} = (1-s)\vec{OA} + s\vec{OD} = (1-s)\vec{a} + s\left(\frac{3}{7}\vec{b}\right) = (1-s)\vec{a} + \frac{3}{7}s\vec{b} \quad \cdots\text{①}$$
> 2. 点 $P$ は線分 $BC$ 上にあるので、 $BP : PC = t : 1-t$ とおける。
>    $$\vec{OP} = t\vec{OC} + (1-t)\vec{OB} = t\left(\frac{2}{5}\vec{a}\right) + (1-t)\vec{b} = \frac{2}{5}t\vec{a} + (1-t)\vec{b} \quad \cdots\text{②}$$
> 3. $\vec{a} \neq \vec{0}, \vec{b} \neq \vec{0}, \vec{a} \nparallel \vec{b}$ より、 $\vec{OP}$ の $\vec{a}, \vec{b}$ による表し方はただ1通りである。
>    よって、①、②の係数を比較する。
>    $$\begin{cases} 1 - s = \frac{2}{5}t \quad \cdots\text{③} \\ \frac{3}{7}s = 1 - t \quad \cdots\text{④} \end{cases}$$
> 4. ③より $5 - 5s = 2t$。④より $3s = 7 - 7t \implies 3s - 7t = -7$。
>    この連立方程式を解くと、 $s = \frac{21}{29}, \ t = \frac{20}{29}$ となる。
> 5. $s = \frac{21}{29}$ を①に代入して、
>    $$\vec{OP} = \mathbf{\frac{8}{29}\vec{a} + \frac{9}{29}\vec{b}}$$
> 
> **解法2：メネラウスの定理によるショートカット（検算に極めて有効）**
> 1. $\triangle OAD$ と直線 $BC$ にメネラウスの定理を適用する。
>    $$\frac{OC}{CA} \times \frac{AP}{PD} \times \frac{DB}{BO} = 1 \implies \frac{2}{3} \times \frac{AP}{PD} \times \frac{4}{7} = 1 \implies \frac{AP}{PD} = \frac{21}{8}$$
> 2. これより、 $AP : PD = 21 : 8$ であることが分かる。
> 3. 分点公式に直接代入する。
>    $$\vec{OP} = \frac{8\vec{OA} + 21\vec{OD}}{21 + 8} = \frac{8\vec{a} + 21\left(\frac{3}{7}\vec{b}\right)}{29} = \mathbf{\frac{8}{29}\vec{a} + \frac{9}{29}\vec{b}}$$

#### [パターン15] 三角形の垂心の位置ベクトル
**【鉄則】垂心 $H$ の条件は「垂直 $\iff$ 内積 $0$」である！ $\vec{AH}\cdot\vec{BC} = 0$ かつ $\vec{BH}\cdot\vec{CA} = 0$ の2式を連立して係数を特定せよ！**

> **【例題 (P.403 基本例題27)】**
> 平面上に $\triangle OAB$ があり、 $OA = 5, \ OB = 6, \ AB = 7$ とする。 $\triangle OAB$ の垂心を $H$ とするとき、 $\vec{OH}$ を $\vec{OA} = \vec{a}, \ \vec{OB} = \vec{b}$ を用いて表せ。
> 
> **【思考プロセスと解答】**
> 1. 準備として、 $\vec{a}\cdot\vec{b}$ の値を求める。
>    $|\vec{b} - \vec{a}|^2 = AB^2 = 49$
>    $|\vec{b}|^2 - 2\vec{a}\cdot\vec{b} + |\vec{a}|^2 = 49 \implies 36 - 2\vec{a}\cdot\vec{b} + 25 = 49 \implies \vec{a}\cdot\vec{b} = 6$
> 2. $\vec{OH} = s\vec{a} + t\vec{b}$ とおく。
> 3. **[条件1： $AH \perp OB$]**
>    $\vec{AH}\cdot\vec{OB} = 0 \iff (\vec{OH} - \vec{OA}) \cdot \vec{OB} = 0$
>    $(s\vec{a} + t\vec{b} - \vec{a}) \cdot \vec{b} = 0 \iff (s-1)\vec{a}\cdot\vec{b} + t|\vec{b}|^2 = 0$
>    $6(s-1) + 36t = 0 \iff s + 6t = 1 \quad \cdots\text{①}$
> 4. **[条件2： $BH \perp OA$]**
>    $\vec{BH}\cdot\vec{OA} = 0 \iff (\vec{OH} - \vec{OB}) \cdot \vec{OA} = 0$
>    $(s\vec{a} + t\vec{b} - \vec{b}) \cdot \vec{a} = 0 \iff s|\vec{a}|^2 + (t-1)\vec{a}\cdot\vec{b} = 0$
>    $25s + 6(t-1) = 0 \iff 25s + 6t = 6 \quad \cdots\text{②}$
> 5. ①, ② を連立方程式として解く。
>    ② $-$ ① より、 $24s = 5 \implies s = \frac{5}{24}$
>    ①に代入して、 $\frac{5}{24} + 6t = 1 \implies 6t = \frac{19}{24} \implies t = \frac{19}{144}$
> 6. よって、 $\vec{OH} = \mathbf{\frac{5}{24}\vec{a} + \frac{19}{144}\vec{b}}$。

#### [パターン16] 三角形の内心・外心の位置ベクトル
**【鉄則】内心 $I$ は「角の二等分線の性質 $BD:DC = c:b$」を用いて内分点を2回適用する！外心 $O$ の証明は「半径が等しい $\iff$ 大きさの2乗が等しい」から内積を計算せよ！**

> **【例題 (授業プリント練習10、P.404基本例題28参考)】**
> 3点 $A, B, C$ が同じ円周上にあり、その中心（外心）を $O$ とする。
> $\vec{OA} + \vec{OB} + \vec{OC} = \vec{0}$ が成り立つとき、 $\triangle ABC$ は正三角形であることを証明せよ。
> 
> **【思考プロセスと解答】**
> 1. 外接円の半径を $r$ とおくと、 $|\vec{OA}| = |\vec{OB}| = |\vec{OC}| = r$ ･･･① である。
> 2. 与えられた等式を変形する。
>    $\vec{OC} = -(\vec{OA} + \vec{OB})$
> 3. 両辺の大きさの2乗をとる。
>    $|\vec{OC}|^2 = |\vec{OA} + \vec{OB}|^2$
>    $|\vec{OC}|^2 = |\vec{OA}|^2 + 2\vec{OA}\cdot\vec{OB} + |\vec{OB}|^2$
> 4. ①を代入する。
>    $r^2 = r^2 + 2\vec{OA}\cdot\vec{OB} + r^2 \iff 2\vec{OA}\cdot\vec{OB} = -r^2 \implies \vec{OA}\cdot\vec{OB} = -\frac{1}{2}r^2$
> 5. なす角の定義 $\cos\angle AOB = \frac{\vec{OA}\cdot\vec{OB}}{|\vec{OA}||\vec{OB}|}$ に代入する。
>    $\cos\angle AOB = \frac{-\frac{1}{2}r^2}{r^2} = -\frac{1}{2}$
>    $0^\circ < \angle AOB < 180^\circ$ より、 $\angle AOB = 120^\circ$。
> 6. 同様の変形により、 $\angle BOC = 120^\circ, \ \angle COA = 120^\circ$ も成り立つ。
> 7. 中心角がすべて $120^\circ$ で等しいため、弦の長さ（辺の長さ）について $AB = BC = CA$ が成り立つ。
> 8. 3辺の長さが等しいため、 $\triangle ABC$ は正三角形である。 （証明終）

#### [パターン17] ベクトルを用いた幾何的等式の証明
**【鉄則】図形の等式の証明は、すべてのベクトルの始点を「適当な1点（通常は $A$ など）」に統一し、内積の展開公式を用いて代数的に計算をすり合わせよ！**

> **【例題 (授業プリント練習8)】**
> $\triangle ABC$ の辺 $BC$ の3等分点のうち、 $B$ に近い方を $D$ とするとき、等式 $2AB^2 + AC^2 = 3(AD^2 + 2BD^2)$ が成り立つことをベクトルを用いて証明せよ。
> 
> **【思考プロセスと解答】**
> 1. 始点を $A$ に揃えてベクトルで表す。
>    $\vec{AB} = \vec{b}, \ \vec{AC} = \vec{c}$ とおく。
> 2. $D$ は $BC$ を $1:2$ に内分する点なので、分点公式より
>    $\vec{AD} = \frac{2\vec{b} + \vec{c}}{3}$
>    また、 $\vec{BD} = \frac{1}{3}\vec{BC} = \frac{1}{3}(\vec{c} - \vec{b})$ である。
> 3. 証明すべき式の左辺と右辺を、それぞれ $\vec{b}, \vec{c}$ を用いて計算する。
>    *   **（左辺）** $= 2|\vec{b}|^2 + |\vec{c}|^2$
>    *   **（右辺）** $= 3\left( |\vec{AD}|^2 + 2|\vec{BD}|^2 \right)$
>        ここで、 $|\vec{AD}|^2 = \left| \frac{2\vec{b} + \vec{c}}{3} \right|^2 = \frac{4|\vec{b}|^2 + 4\vec{b}\cdot\vec{c} + |\vec{c}|^2}{9}$
>        $|\vec{BD}|^2 = \left| \frac{\vec{c} - \vec{b}}{3} \right|^2 = \frac{|\vec{b}|^2 - 2\vec{b}\cdot\vec{c} + |\vec{c}|^2}{9}$
>        これらを右辺に代入して整理する。
>        $(\text{右辺}) = 3 \left( \frac{4|\vec{b}|^2 + 4\vec{b}\cdot\vec{c} + |\vec{c}|^2}{9} + 2 \cdot \frac{|\vec{b}|^2 - 2\vec{b}\cdot\vec{c} + |\vec{c}|^2}{9} \right)$
>        $= \frac{3}{9} \left( (4|\vec{b}|^2 + 4\vec{b}\cdot\vec{c} + |\vec{c}|^2) + (2|\vec{b}|^2 - 4\vec{b}\cdot\vec{c} + 2|\vec{c}|^2) \right)$
>        $= \frac{1}{3} \left( 6|\vec{b}|^2 + 3|\vec{c}|^2 \right) = 2|\vec{b}|^2 + |\vec{c}|^2$
> 4. 計算の結果、（左辺）$=$（右辺）となり、等式は成り立つ。 （証明終）

#### [パターン18] 内積と三角形の形状決定
**【鉄則】「どんな三角形か」を問われたら、等式のベクトルをすべて「始点 $A$」に統一（分解）し、移項して「因数分解（内積の形）」を作り出し、垂直または長さの等式を導け！**

> **【例題 (P.413 基本例題33)】**
> $\triangle ABC$ が次の関係式を満たすとき、どのような形の三角形か。
> $(\vec{AB} - \vec{AC}) \cdot \vec{BC} = |\vec{AC}|^2$  *(※類似問題として考え方を記述)*
> 
> **【思考プロセスと解答】**
> 1. すべてのベクトルの始点を $A$ に揃える。
>    $\vec{AB} = \vec{b}, \ \vec{AC} = \vec{c}$ とおく。 $\vec{BC} = \vec{c} - \vec{b}$ である。
> 2. 与式に代入して整理する。
>    $(\vec{b} - \vec{c}) \cdot (\vec{c} - \vec{b}) = |\vec{c}|^2$
>    $-|\vec{b} - \vec{c}|^2 = |\vec{c}|^2 \iff -|\vec{BC}|^2 = |\vec{AC}|^2$ ･･･これは幾何的に矛盾。
>    **【正しい問題のパターン（例： $\vec{AB}\cdot\vec{BC} = -|\vec{AB}|^2$ など）】**
>    始点を $A$ に揃えて展開すると、 $\vec{b}\cdot(\vec{c} - \vec{b}) = -|\vec{b}|^2 \iff \vec{b}\cdot\vec{c} - |\vec{b}|^2 = -|\vec{b}|^2 \iff \vec{b}\cdot\vec{c} = 0$。
>    これより、 $\vec{AB}\cdot\vec{AC} = 0 \iff \vec{AB} \perp \vec{AC}$ が導かれる。
>    したがって、 **$\angle A = 90^\circ$ の直角三角形** と判断できる。

#### [パターン12の発展] 交点の延長線と対辺の交点（係数の和＝1の利用）
**【鉄則】交点 $P$ の位置ベクトル $\vec{OP}$ が求まっているとき、直線 $OP$ の延長線と辺 $AB$ の交点 $Q$ を求めるには、①「$Q$ は $OP$ の延長線上 $\iff \vec{OQ} = k\vec{OP}$」、②「$Q$ は $AB$ 上 $\iff$ 係数の和が $1$」の2ステップで瞬殺せよ！**

> **【例題 (授業プリント Q55、青チャートP.402・405参考)】**
> $\triangle OAB$ において、交点 $P$ の位置ベクトルが $\vec{OP} = \frac{2}{13}\vec{OA} + \frac{3}{13}\vec{OB}$ であるとする。直線 $OP$ と辺 $AB$ の交点を $Q$ とするとき、
> (1) $\vec{OQ}$ を $\vec{OA} = \vec{a}, \ \vec{OB} = \vec{b}$ を用いて表せ。
> (2) 比 $AQ : QB$ および $OP : PQ$ を求めよ。
> 
> **【思考プロセスと解答】**
> (1)
> 1. 点 $Q$ は直線 $OP$ 上にあるので、実数 $k$ を用いて $\vec{OQ} = k\vec{OP}$ と表せる。
>    $$\vec{OQ} = k\left( \frac{2}{13}\vec{a} + \frac{3}{13}\vec{b} \right) = \frac{2k}{13}\vec{a} + \frac{3k}{13}\vec{b} \quad \cdots\text{①}$$
> 2. 点 $Q$ は辺 $AB$ 上にあるので、 $\vec{a}$ と $\vec{b}$ の**係数の和が $1$** になる。
>    $$\frac{2k}{13} + \frac{3k}{13} = 1 \iff \frac{5k}{13} = 1 \implies k = \frac{13}{5}$$
> 3. $k = \frac{13}{5}$ を①に代入する。
>    $$\vec{OQ} = \frac{13}{5} \left( \frac{2}{13}\vec{a} + \frac{3}{13}\vec{b} \right) = \mathbf{\frac{2}{5}\vec{a} + \frac{3}{5}\vec{b}}$$
> 
> (2)
> 1. (1)より、 $\vec{OQ} = \frac{2\vec{a} + 3\vec{b}}{5}$ である。
>    これは、点 $Q$ が線分 $AB$ を **$3:2$ に内分する点**であることを意味する。
>    よって、 $\mathbf{AQ : QB = 3 : 2}$。
> 2. $\vec{OQ} = k\vec{OP}$ の関係から、 $\vec{OQ} = \frac{13}{5}\vec{OP} \iff \vec{OP} = \frac{5}{13}\vec{OQ}$。
>    これより、線分 $OQ$ の長さを $13$ とすると、 $OP = 5$ である。
>    よって、 $PQ = 13 - 5 = 8$ となり、 $\mathbf{OP : PQ = 5 : 8}$。

## 1. 図形と方程式の基本要綱

### 1.1 座標平面上の点
**① 2点間の距離**
*   2点 $A(x_1, y_1), B(x_2, y_2)$ 間の距離 $AB$：
    $$AB = \sqrt{(x_2-x_1)^2 + (y_2-y_1)^2}$$
    （※ 始点を原点 $O(0, 0)$ とすると、 $OA = \sqrt{x_1^2 + y_1^2}$）

**② 内分点・外分点・重心の座標**
2点 $A(x_1, y_1), B(x_2, y_2)$ を結ぶ線分 $AB$ について、
*   **内分点**： $m:n$ に内分する点 $P(x, y)$
    $$x = \frac{nx_1 + mx_2}{m+n}, \quad y = \frac{ny_1 + my_2}{m+n}$$
*   **外分点**： $m:n$ に外分する点 $Q(x, y)$
    $$x = \frac{-nx_1 + mx_2}{m-n}, \quad y = \frac{-ny_1 + my_2}{m-n}$$
    *(※ 外分は「$-n$ に内分する」と捉えると、内分の公式の $n$ を $-n$ に置き換えるだけで導ける。)*
*   **重心**： $\triangle ABC$ の重心 $G(x, y)$
    $$x = \frac{x_1+x_2+x_3}{3}, \quad y = \frac{y_1+y_2+y_3}{3}$$

---

### 1.2 直線の方程式と2直線の関係
**① 直線の方程式**
*   点 $(x_1, y_1)$ を通り、傾き $m$ の直線：
    $$y - y_1 = m(x - x_1)$$
*   異なる2点 $A(x_1, y_1), B(x_2, y_2)$ を通る直線：
    *   $x_1 \neq x_2$ のとき： $y - y_1 = \frac{y_2 - y_1}{x_2 - x_1}(x - x_1)$
    *   $x_1 = x_2$ のとき： $x = x_1$
*   $x$ 切片が $a$、$y$ 切片が $b$ （$a \neq 0, b \neq 0$）の直線：
    $$\frac{x}{a} + \frac{y}{b} = 1$$

**② 2直線の関係**
2直線 $y = m_1x + n_1, \ y = m_2x + n_2$ について、
*   **平行条件**： $m_1 = m_2$
*   **垂直条件**： $m_1m_2 = -1$

一般形 $a_1x + b_1y + c_1 = 0$ と $a_2x + b_2y + c_2 = 0$ について、
*   **平行条件**： $\mathbf{a_1b_2 - a_2b_1 = 0}$
*   **垂直条件**： $\mathbf{a_1a_2 + b_1b_2 = 0}$
*(※ 文字定数を含む問題では、分母に文字がくるのを防ぐため、必ず一般形の条件式を使用すること。)*

**③ 点と直線の距離**
*   点 $(x_1, y_1)$ と直線 $ax + by + c = 0$ の距離 $d$：
    $$d = \frac{|ax_1 + by_1 + c|}{\sqrt{a^2 + b^2}}$$
    （※ 原点 $(0,0)$ との距離は $d = \frac{|c|}{\sqrt{a^2+b^2}}$）

---

### 1.3 円の方程式
**① 円の方程式**
*   **標準形**：中心 $(a, b)$、半径 $r$ の円
    $$(x-a)^2 + (y-b)^2 = r^2$$
*   **一般形**： $x^2 + y^2 + lx + my + n = 0$
    *(※一般形は平方完成によって中心と半径を特定する。)*

**② 円と直線の位置関係**
円の中心と直線の距離を $d$、円の半径を $r$ とすると、
*   $d < r \iff$ 異なる2点で交わる
*   $d = r \iff$ 1点で接する（接する）
*   $d > r \iff$ 共有点をもたない

**③ 円の接線の方程式**
*   円 $x^2 + y^2 = r^2$ 上の点 $(x_1, y_1)$ における接線：
    $$\mathbf{x_1x + y_1y = r^2}$$
*   円 $(x-a)^2 + (y-b)^2 = r^2$ 上の点 $(x_1, y_1)$ における接線：
    $$\mathbf{(x_1-a)(x-a) + (y_1-b)(y-b) = r^2}$$

**④ 2つの円の位置関係**
中心間の距離を $d$、2つの円の半径を $r, r'$ ($r > r'$) とするとき、外接・内接・交わるなどの判定は $d$ と $r \pm r'$ の大小関係で決まる。（※ 既出のため詳細省略）

**⑤ 2つの円の交点を通る図形の方程式（円束）**
2つの円 $x^2+y^2+lx+my+n=0$, $x^2+y^2+l'x+m'y+n'=0$ が2点で交わるとき、方程式
$$(x^2+y^2+lx+my+n) + k(x^2+y^2+l'x+m'y+n') = 0$$
は、
*   $k = -1$ のとき、2つの円の共有点を通る**直線（共通弦）**
*   $k \neq -1$ のとき、2つの円の共有点を通る**円**
を表す。

## 2. 最難関・定番解法パターン

#### [パターン19] 2直線の交点を通る直線（束の活用）
**【鉄則】「2直線の交点を通る新たな直線」は、交点を連立方程式で求めてはならない。恒等式 $k(l_1) + l_2 = 0$ を立てて代入せよ！**

> **【例題】**
> 2直線 $2x - y + 3 = 0$ ･･･① と $3x + 2y - 1 = 0$ ･･･② の交点を通り、点 $(2, 4)$ を通る直線の方程式を求めよ。
> 
> **【思考プロセスと解答】**
> 1. ①、②の交点を通る直線の方程式を、定数 $k$ を用いて表す。
>    $$k(2x - y + 3) + (3x + 2y - 1) = 0 \quad \cdots\text{③}$$
> 2. ③の直線が点 $(2, 4)$ を通ることから、 $x=2, y=4$ を代入する [10].
>    $$k(2 \cdot 2 - 4 + 3) + (3 \cdot 2 + 2 \cdot 4 - 1) = 0$$
>    $$3k + 13 = 0 \implies k = -\frac{13}{3}$$
> 3. 求めた $k$ を③に代入して整理する。
>    $$-\frac{13}{3}(2x - y + 3) + (3x + 2y - 1) = 0$$
>    $$-13(2x - y + 3) + 3(3x + 2y - 1) = 0$$
>    $$-26x + 13y - 39 + 9x + 6y - 3 = 0 \iff -17x + 19y - 42 = 0$$
> 4. よって、求める直線の方程式は **$17x - 19y + 42 = 0$**。

#### [パターン20] 直線に関して対称な直線の方程式
**【鉄則】「直線 $l$ に関して直線 $m$ と対称な直線 $m'$」を求める際は、①「$m$ と $l$ の交点を出す（$m'$ もこの点を通る）」、②「$m$ 上の任意の点の対称点を $l$ に関して求める」の2点を通る直線として解け！**

> **【例題 (章末問題類題)】**
> 直線 $l: x - y - 1 = 0$ に関して、直線 $m: x - 2y + 3 = 0$ と対称な直線 $m'$ の方程式を求めよ。
> 
> **【思考プロセスと解答】**
> 1. **[ステップ1：交点を求める]**
>    $m$ と $l$ の交点 $A$ を連立方程式で求める。
>    $\begin{cases} x - 2y + 3 = 0 \\ x - y - 1 = 0 \end{cases} \implies y = 4, \ x = 5 \implies A(5, 4)$
>    この交点 $A(5, 4)$ は、求める対称な直線 $m'$ 上にも存在する。
> 2. **[ステップ2：代表点を選ぶ]**
>    直線 $m$ 上の計算しやすい点を1つ選ぶ。 $y=0$ とすると $x=-3$ となるため、点 $Q(-3, 0)$ を選ぶ。
> 3. **[ステップ3：対称点を求める]**
>    点 $Q(-3, 0)$ の、直線 $l: x - y - 1 = 0$ に関して対称な点 $Q'(x_0, y_0)$ を求める。
>    *   **垂直条件**： $QQ' \perp l$ より、 $l$ の傾きが $1$ なので、直線 $QQ'$ の傾きは $-1$。
>        $$\frac{y_0 - 0}{x_0 - (-3)} = -1 \iff x_0 + y_0 + 3 = 0 \quad \cdots\text{④}$$
>    *   **中点条件**： 線分 $QQ'$ の中点 $\left(\frac{x_0-3}{2}, \frac{y_0}{2}\right)$ は直線 $l$ 上にある。
>        $$\frac{x_0-3}{2} - \frac{y_0}{2} - 1 = 0 \iff x_0 - y_0 - 5 = 0 \quad \cdots\text{⑤}$$
>    *   ④, ⑤を連立して解くと、 $x_0 = 1, \ y_0 = -4$。 よって、 $Q'(1, -4)$。
> 4. **[ステップ4：2点を通る直線の方程式]**
>    求める直線 $m'$ は、2点 $A(5, 4)$ と $Q'(1, -4)$ を通る直線である。
>    傾きは $\frac{-4 - 4}{1 - 5} = 2$。
>    $$y - 4 = 2(x - 5) \iff 2x - y - 6 = 0$$
> 5. よって、求める直線の方程式は **$2x - y - 6 = 0$**。

#### [パターン21] 2つの円の共通接線の方程式
**【鉄則】共通接線を求めるときは、「接点を文字 $(p, q)$ で置く」のが最も速い！ 接線公式 $px+qy=r^2$ を作り、もう1つの円の中心との距離が「半径 $r'$」に等しいとして処理せよ！**

> **【例題 (問題集P.045、C問題応用)】**
> 2つの円 $C_1: x^2 + y^2 = 4$ と $C_2: (x-4)^2 + y^2 = 1$ の共通接線の方程式をすべて求めよ。
> 
> **【思考プロセスと解答】**
> 1. 求める接線が円 $C_1$ と接する点（接点）を $P(p, q)$ とおく。
>    点 $P$ は円 $C_1$ 上にあるので、 **$p^2 + q^2 = 4$** ･･･⑥ が成り立つ。
> 2. 点 $P$ における円 $C_1$ の接線の方程式は、 **$px + qy = 4 \iff px + qy - 4 = 0$** ･･･⑦ である。
> 3. この直線 ⑦ が、もう1つの円 $C_2$（中心 $(4, 0)$、半径 $1$）にも接する条件を立てる。
>    円 $C_2$ の中心 $(4,0)$ と直線 ⑦ との距離が、半径 $1$ に等しければよい。
>    $$\frac{|p \cdot 4 + q \cdot 0 - 4|}{\sqrt{p^2 + q^2}} = 1$$
> 4. ⑥より $\sqrt{p^2+q^2} = \sqrt{4} = 2$ を代入する。
>    $$\frac{|4p - 4|}{2} = 1 \iff |2p - 2| = 1$$
> 5. これを解いて、 $p$ の値を特定する。
>    *   $2p - 2 = 1 \implies p = \frac{3}{2}$
>    *   $2p - 2 = -1 \implies p = \frac{1}{2}$
> 6. それぞれの場合の $q$ の値を ⑥ から求める。
>    *   $p = \frac{3}{2}$ のとき： $q^2 = 4 - \frac{9}{4} = \frac{7}{4} \implies q = \pm \frac{\sqrt{7}}{2}$
>    *   $p = \frac{1}{2}$ のとき： $q^2 = 4 - \frac{1}{4} = \frac{15}{4} \implies q = \pm \frac{\sqrt{15}}{2}$
> 7. 得られた $p, q$ の組を⑦の接線の式に代入して整理する。
>    *   $\frac{3}{2}x \pm \frac{\sqrt{7}}{2}y = 4 \iff \mathbf{3x \pm \sqrt{7}y = 8}$
>    *   $\frac{1}{2}x \pm \frac{\sqrt{15}}{2}y = 4 \iff \mathbf{x \pm \sqrt{15}y = 8}$
> 8. よって、共通接線は全部で **$3x \pm \sqrt{7}y = 8, \ x \pm \sqrt{15}y = 8$** の4本である。
> *(※ 接線 $y=mx+n$ とおくと、 $y$ 軸平行な接線 $x=\pm 2$ の場合分けが発生するが、接点 $(p,q)$ で置くこの解法はすべてを自動で含んで解けるため、最難関入試で極めて強力です。)*

## 第2部：図形と方程式（軌跡と領域） 第2章
**〜 軌跡と方程式の極意（連動点・パラメータ消去・除外点） 〜**

---

## 1. 軌跡と方程式の基本手順

**① 軌跡とは**
*   条件を満たす点全体が描く図形のこと。

**② 軌跡を求める3ステップ（王道手順）**
1.  **動点の設定**：
    条件を満たして動く点（軌跡を描く点）を $P(x, y)$ とおく。
2.  **関係式の導出（パラメータ消去）**：
    問題文の条件を $x, y$ の等式で表す。他に動く点 $Q(s, t)$ やパラメータ $t$ がある場合は、それらを消去して $x, y$ だけの式に帰着させる。
3.  **十分性の確認（除外点の検討）**：
    得られた図形（円や放物線など）の**すべて**が条件を満たすか確認する。特に、文字による割り算を行った場合や、動点に範囲制限がある場合は、**「除外点」や「定義域の制限」**が発生するため、厳密な確認が必要である。

---

## 2. 最難関・定番解法パターン

#### [パターン22] 2点からの距離の比が一定な点の軌跡（アポロニウスの円）
**【鉄則】「2点からの距離の比が $m:n$ （$m \neq n$）」となる点の軌跡は、等式 $AP : BP = m : n \iff n^2 AP^2 = m^2 BP^2$ を立て、座標計算で円の方程式に整理せよ！**

> **【例題 (P.043 Q206)】**
> 2点 $A(-2, 0), B(4, 0)$ からの距離の比が $2:1$ である点 $P$ の軌跡を求めよ。
> 
> **【思考プロセスと解答】**
> 1. 求める点 $P$ の座標を $(x, y)$ とおく。
> 2. 条件より、 $AP : BP = 2 : 1 \iff AP = 2BP$
> 3. 両辺を2乗してルートを外す。
>    $$AP^2 = 4BP^2$$
> 4. 2点間の距離の公式を用いて、 $x, y$ の等式にする。
>    $$(x+2)^2 + y^2 = 4 \left\{ (x-4)^2 + y^2 \right\}$$
> 5. 展開して整理する。
>    $$x^2 + 4x + 4 + y^2 = 4(x^2 - 8x + 16 + y^2)$$
>    $$3x^2 - 36x + 3y^2 + 60 = 0$$
>    両辺を $3$ で割る。
>    $$x^2 - 12x + y^2 + 20 = 0$$
> 6. 平方完成して、円の標準形にする。
>    $$(x-6)^2 + y^2 = 16$$
> 7. よって、求める軌跡は、 **中心が点 $(6, 0)$、半径が $4$ の円** である。

#### [パターン23] 連動して動く点の軌跡（媒介変数 $s, t$ の消去）
**【鉄則】「他の動く点 $Q(s, t)$ に連動して動く点 $P(x, y)$」の軌跡は、①「$Q$ が満たす関係式 $f(s, t) = 0$」を立て、②「$s, t$ を $x, y$ の式で表して代入」せよ！**

> **【例題 (P.092 練習2(1)類題)】**
> 点 $A(4, 1)$ と、放物線 $y = x^2$ 上を動く点 $Q$ がある。線分 $AQ$ を $1:2$ に内分する点 $P$ の軌跡を求めよ。
> 
> **【思考プロセスと解答】**
> 1. 動く点 $Q$ の座標を $(s, t)$ とおき、軌跡を描く点 $P$ の座標を $(x, y)$ とおく。
> 2. 点 $Q(s, t)$ は放物線 $y = x^2$ 上を動くので、次の関係式が成り立つ。
>    $$t = s^2 \quad \cdots\text{①}$$
> 3. 点 $P(x, y)$ は、 $A(4, 1)$ と $Q(s, t)$ を $1:2$ に内分する点なので、分点公式より
>    $$x = \frac{2 \cdot 4 + 1 \cdot s}{1+2} = \frac{8+s}{3} \iff s = 3x - 8 \quad \cdots\text{②}$$
>    $$y = \frac{2 \cdot 1 + 1 \cdot t}{1+2} = \frac{2+t}{3} \iff t = 3y - 2 \quad \cdots\text{③}$$
> 4. ②、③を①に代入して、 $s, t$ を消去する。
>    $$3y - 2 = (3x - 8)^2$$
>    $$3y = 9\left(x - \frac{8}{3}\right)^2 + 2 \iff y = 3\left(x - \frac{8}{3}\right)^2 + \frac{2}{3}$$
> 5. よって、求める軌跡は、 **放物線 $y = 3\left(x - \frac{8}{3}\right)^2 + \frac{2}{3}$** （または $y = 3x^2 - 16x + 22$）である。

#### [パターン24] 直交する直線群の交点の軌跡と「除外点」
**【鉄則】2直線からパラメータ $m$ を消去するとき、 $x \neq 0$ と仮定して $m = \frac{\cdots}{x}$ を代入する。このとき、割り算の前提から漏れた「$x = 0$ のときの点」が軌跡上に存在するかを個別にチェックし、除外点を厳密に特定せよ！**

> **【例題 (P.044 例題15)】**
> 実数 $m$ が変化するとき、次の2直線の交点 $P$ の軌跡を求めよ。
> $mx - y + 1 = 0 \ \cdots\text{①}, \quad x + m(y-1) - 2 = 0 \ \cdots\text{②}$
> 
> **【思考プロセスと解答】**
> 1. ①、②から $m$ を消去したい。①より $mx = y - 1$ である。
> 2. **[場合分け1： $x \neq 0$ のとき]**
>    $m = \frac{y-1}{x}$ と表せる。これを②に代入する。
>    $$x + \frac{y-1}{x}(y-1) - 2 = 0$$
>    両辺に $x$ を掛けて整理する。
>    $$x^2 + (y-1)^2 - 2x = 0 \iff (x-1)^2 + (y-1)^2 = 1 \quad \cdots\text{③}$$
> 3. **[場合分け2： $x = 0$ のとき]**
>    ①に $x=0$ を代入すると、 $-y+1=0 \implies y=1$。
>    この点 $(0, 1)$ が、もう1つの直線②上にあるか（交点になり得るか）をチェックする。
>    ②の左辺に $x=0, y=1$ を代入すると、
>    $$0 + m(1-1) - 2 = -2 \neq 0$$
>    となり、等式は成り立たない。すなわち、点 $(0, 1)$ は交点 $P$ にはなり得ない。
> 4. **[結論の構築]**
>    交点 $P$ は、円③の上を動くが、 $x = 0$ のときの点、すなわち③に $x=0$ を代入して得られる点 $(0, 1)$ と $(0, 1)$ （※重複、実際は $(x-1)^2 + (y-1)^2 = 1$ に $x=0$ を代入すると $(y-1)^2=0 \implies y=1$。点 $(0,1)$ のみ）は除外される。
>    よって、求める軌跡は、 **円 $(x-1)^2 + (y-1)^2 = 1$。ただし、点 $(0, 1)$ は除く**。

#### [パターン25] パラメータ制限を伴う軌跡（放物線の頂点と隠れた定義域）
**【鉄則】放物線の頂点の軌跡や、媒介変数（パラメータ）に制限がある問題では、パラメータを消去した後に、必ず「パラメータの制限範囲」から「$x$ の定義域（制限）」を逆算して図示せよ！**

> **【例題 (P.044 例題14)】**
> $x = -t + 3, \ y = t^2 - 2$ とする。実数 $t$ が $t > 0$ の範囲を動くとき、点 $(x, y)$ の軌跡を求めよ。
> 
> **【思考プロセスと解答】**
> 1. $x = -t + 3$ より、 $t = -x + 3$ ･･･① とする。
> 2. ①を $y = t^2 - 2$ に代入して $t$ を消去する。
>    $$y = (-x + 3)^2 - 2 \iff y = x^2 - 6x + 7 \quad \cdots\text{②}$$
> 3. **【隠れた定義域の確認】**
>    パラメータに $t > 0$ という制限がある。①より $t = -x + 3$ なので、
>    $$-x + 3 > 0 \iff \mathbf{x < 3}$$
>    という $x$ の範囲制限（定義域）が生まれる。
> 4. よって、求める軌跡は、 **放物線 $y = x^2 - 6x + 7$ の $x < 3$ の部分** である。

#### [追加パターンA] 角の二等分線の方程式（等距離の等式）
**【鉄則】「2直線 $l_1, l_2$ のなす角の二等分線」 $\iff$ 「2直線からの距離が等しい点 $P(x, y)$ の軌跡（ $d_1 = d_2$ ）」！ 分母の有理化（整理）を行って、絶対値記号を $A = \pm B$ で外せ！**

> **【例題 (軌跡と領域PDF Q6(1))】**
> 2直線 $l_1: x - 2y - 2 = 0$ と $l_2: 4x - 2y + 1 = 0$ のなす角の二等分線の方程式を求めよ。
> 
> **【思考プロセスと解答】**
> 1. 求める二等分線上の動点を $P(x, y)$ とおく。
> 2. 点 $P$ から直線 $l_1$ までの距離を $d_1$、 $l_2$ までの距離を $d_2$ とすると、 $d_1 = d_2$ が成り立つ。
>    $$\frac{|x - 2y - 2|}{\sqrt{1^2 + (-2)^2}} = \frac{|4x - 2y + 1|}{\sqrt{4^2 + (-2)^2}}$$
> 3. 分母を整理する。 $\sqrt{1^2 + (-2)^2} = \sqrt{5}$、 $\sqrt{4^2 + (-2)^2} = \sqrt{20} = 2\sqrt{5}$。
>    $$\frac{|x - 2y - 2|}{\sqrt{5}} = \frac{|4x - 2y + 1|}{2\sqrt{5}}$$
> 4. 両辺に $2\sqrt{5}$ を掛けて分母を払う。
>    $$2|x - 2y - 2| = |4x - 2y + 1| \iff |2x - 4y - 4| = |4x - 2y + 1|$$
> 5. 絶対値記号を $|A| = |B| \iff A = \pm B$ を用いて外す（2通りに場合分け）。
>    *   **（i） $2x - 4y - 4 = 4x - 2y + 1$ のとき**
>        $$-2x - 2y - 5 = 0 \iff 2x + 2y + 5 = 0$$
>    *   **（ii） $2x - 4y - 4 = -(4x - 2y + 1)$ のとき**
>        $$2x - 4y - 4 = -4x + 2y - 1 \iff 6x - 6y - 3 = 0 \iff 2x - 2y - 1 = 0$$
> 6. よって、求める角の二等分線の方程式は、互いに直交する2直線 **$2x + 2y + 5 = 0, \ 2x - 2y - 1 = 0$** である。


# 数学まとめプリント 第7回
## 第2部：図形と方程式（軌跡と領域） 第3章
**〜 不等式の表す領域と領域における最大・最小（線形・非線形計画法） 〜**

---

## 1. 不等式の表す領域

### 1.1 基本事項・要点のまとめ
**① 直線が分ける領域**
境界線となる直線を $y = mx + n$ とする。
*   不等式 $\mathbf{y > mx + n}$ の表す領域は、直線の上側の部分。
*   不等式 $\mathbf{y < mx + n}$ の表す領域は、直線の下側の部分。
*   一般に、不等式 $ax + by + c > 0$ （$b > 0$）は、 $y > -\frac{a}{b}x - \frac{c}{b}$ と変形できるため、直線の**上側**を表す。（※ $b < 0$ のときは不等号の向きが逆転し、**下側**となる）

**② 円が分ける領域**
境界線となる円を $(x-a)^2 + (y-b)^2 = r^2$ とする。
*   不等式 $\mathbf{(x-a)^2 + (y-b)^2 < r^2}$ の表す領域は、円の**内部**。
*   不等式 $\mathbf{(x-a)^2 + (y-b)^2 > r^2}$ の表す領域は、円の**外部**。

**③ 連立不等式の表す領域**
*   複数の不等式を同時に満たす領域。それぞれの不等式が表す領域の共通部分（重なり合う部分）を塗り分ける。

**④ 領域を用いた等式・不等式の証明（境界値と包含関係）**
*   条件 $p \implies q$ が真であることは、条件 $p$ を満たす点全体の集合 $P$ と、条件 $q$ を満たす点全体の集合 $Q$ について、包含関係 $\mathbf{P} \subset \mathbf{Q}$ が成り立つことと同値である。

---

## 2. 最難関・定番解法パターン

#### [パターン26] 絶対値を含む不等式の表す領域
**【鉄則】絶対値記号を含む不等式は、①「場合分けをして絶対値を外す」か、②「図形の対称性（$x \leftrightarrow -x, \ y \leftrightarrow -y$ など）」を利用して第1象限のみを描き、それを折り返して全体を完成させよ！**

> **【例題 (P.045 Q214, Q227)】**
> 不等式 $|x| + 2|y| \leqq 4$ の表す領域を図示せよ。
> 
> **【思考プロセスと解答】**
> 1. 式の対称性に注目する。 $x$ を $-x$ に、 $y$ を $-y$ に置き換えても、絶対値の性質 $|-x| = |x|$ より式は変わらない。
>    よって、この領域は **$x$ 軸、 $y$ 軸、および原点に関して対称**である。
> 2. 第1象限（$x \geqq 0, y \geqq 0$）のみを考える。このとき絶対値はそのまま外れる。
>    $$x + 2y \leqq 4 \iff y \leqq -\frac{1}{2}x + 2$$
> 3. 第1象限において、直線 $y = -\frac{1}{2}x + 2$ の下側（軸を含む）を作図する。
> 4. この図形を、 $y$ 軸に関して対称移動して第2象限へ、さらに $x$ 軸に関して対称移動して第3・第4象限へ折り返す。
> 5. 領域は、4点 $(4, 0), (0, 2), (-4, 0), (0, -2)$ を頂点とする**ひし形の内部および周**となる。（境界線を含む）

#### [パターン27] 積の形 $f(x, y)g(x, y) > 0$ が表す領域
**【鉄則】積が正 $\iff$ 「(共に正) または (共に負)」！ 境界線を描いた後、適当な代表点（原点など）を代入して不等式が成り立つかを確認し、チェック柄のように交互に領域を塗り分けよ！**

> **【例題 (P.045 Q218、P.046 Q221)】**
> 不等式 $(2x + 3y + 6)(x - 2y + 1) \geqq 0$ の表す領域を図示せよ。
> 
> **【思考プロセスと解答】**
> 1. 不等式が成り立つ条件を2つのケースに分解する。
>    $$\begin{cases} 2x + 3y + 6 \geqq 0 \\ x - 2y + 1 \geqq 0 \end{cases} \quad\text{または}\quad \begin{cases} 2x + 3y + 6 \leqq 0 \\ x - 2y + 1 \leqq 0 \end{cases}$$
> 2. それぞれの直線を境界線として描く。
>    $l_1: 2x+3y+6=0 \iff y = -\frac{2}{3}x - 2$
>    $l_2: x-2y+1=0 \iff y = \frac{1}{2}x + \frac{1}{2}$
> 3. **【境界値チェック法】**
>    境界線上にない代表点として原点 $(0, 0)$ を選び、元の等式の左辺に代入する。
>    $$(2\cdot 0 + 3\cdot 0 + 6)(0 - 2\cdot 0 + 1) = 6 \cdot 1 = 6 \geqq 0$$
>    等式が成り立つため、**原点 $(0, 0)$ を含む領域は塗り分ける対象**となる。
> 4. 2直線によって平面は4つの領域に分割されるが、積の性質上、隣り合う領域は「正」と「負」が交互に並ぶ。
> 5. 原点を含む領域、およびその対頂角に位置する領域の2箇所を塗りつぶす。
> 6. 求める領域は、下図の斜線部分（境界線を含む）。

#### [パターン28] 領域における最大・最小①（線形計画法）
**【鉄則】領域内を動く点 $(x, y)$ に対して $ax + by$ の最大・最小を求めるときは、 $ax + by = k \iff y = -\frac{a}{b}x + \frac{k}{b}$ とおき、傾き $-\frac{a}{b}$ の直線が「領域と共有点をもつ」条件から、 $y$ 切片の最大・最小をグラフ上で特定せよ！**

> **【例題 (不等式の表す領域 PDF 練習8、問題集P.046 Q222)】**
> ある工場で製品A, Bを製造している。1トン製造するのに必要な原料P, Qの量、および1トンあたりの利益は下表の通りである。
> | | 原料P | 原料Q | 1トンあたりの利益 |
> | :--- | :---: | :---: | :---: |
> | 製品A | 3トン | 5トン | 40万円 |
> | 製品B | 6トン | 2トン | 30万円 |
> 1か月間に使用できる原料Pは最大120トン、原料Qは最大80トンである。利益を最大にするには、製品A, Bをそれぞれ毎月何トンずつ製造すればよいか。
> 
> **【思考プロセスと解答】**
> 1. 製造する製品Aを $x$ トン、製品Bを $y$ トンとおく。
> 2. 条件（原料の制約）を不等式で表す。 $x, y$ は生産量なので $x \ge 0, y \ge 0$ [10]。
>    *   原料Pの制約： $3x + 6y \leqq 120 \iff x + 2y \leqq 40$
>    *   原料Qの制約： $5x + 2y \leqq 80$
> 3. この4つの不等式が表す連立領域を描く。
>    境界線 $x+2y=40$ と $5x+2y=80$ の交点を連立して求めると、 $(10, 15)$ となる。
> 4. 最大化したい総利益を $k$ 万円とおく。
>    $$40x + 30y = k \iff y = -\frac{4}{3}x + \frac{k}{30}$$
> 5. 傾き $-\frac{4}{3} \approx -1.33$ の直線を、描いた領域と交わるように動かす。
>    境界となる2直線の傾きは、 $x+2y=40 \implies -0.5$、 $5x+2y=80 \implies -2.5$ である。
>    傾き $-\frac{4}{3}$ はこの2つの間にあるため、直線が**交点 $(10, 15)$ を通るときに $y$ 切片 $\frac{k}{30}$（すなわち利益 $k$）は最大**となる。
> 6. $x=10, y=15$ を代入して、最大値 $k$ を求める。
>    $$k = 40(10) + 30(15) = 400 + 450 = 850 \text{ 万円}$$
> 7. よって、**製品Aを10トン、製品Bを15トン**製造するとき、最大利益 **850万円** を得る。

#### [パターン29] 領域における最大・最小②（非線形計画法）
**【鉄則】円 $x^2 + y^2 = k$ は「原点からの距離の2乗」、 $\frac{y}{x} = k$ は「原点を通る直線の傾き」を意味する！ 式の形がもつ図形的意味を見破って解け！**

> **【例題 (不等式の表す領域 PDF 練習9)】**
> 点 $(x, y)$ が領域 $(x-3)^2 + (y-2)^2 \leqq 1$ 内を動くとき、次の式の最大値と最小値を求めよ。
> (1) $x^2 + y^2$
> (2) $\frac{y}{x}$
> 
> **【思考プロセスと解答】**
> 動点 $P(x, y)$ が動く領域は、中心 $A(3, 2)$、半径 $1$ の円の内部および周である。
> 
> (1) $x^2 + y^2$ は、**原点 $O(0,0)$ と点 $P(x,y)$ の距離 $OP$ の2乗**を表す。
> 1. 原点 $O$ と円の中心 $A(3, 2)$ の距離 $OA$ を求める。
>    $$OA = \sqrt{3^2 + 2^2} = \sqrt{13}$$
> 2. 同一直線上で、中心からさらに半径分進んだとき（$OA + r$）に距離は最大になり、手前で止まったとき（$OA - r$）に最小になる。
>    *   最大距離： $\sqrt{13} + 1 \implies$ 最大値は $(\sqrt{13} + 1)^2 = \mathbf{14 + 2\sqrt{13}}$
>    *   最小距離： $\sqrt{13} - 1 \implies$ 最小値は $(\sqrt{13} - 1)^2 = \mathbf{14 - 2\sqrt{13}}$
> 
> (2) $\frac{y}{x}$ は、**原点 $O(0,0)$ と点 $P(x,y)$ を結ぶ直線の傾き**を表す。
> 1. $\frac{y}{x} = k \iff y = kx \iff kx - y = 0$ とおく。
> 2. この直線が円 $(x-3)^2 + (y-2)^2 \leqq 1$ と共有点をもつ、すなわち「接するとき（直線までの距離 $d = 1$）」に傾き $k$ は最大・最小となる。
>    $$\frac{|k \cdot 3 - 2|}{\sqrt{k^2 + (-1)^2}} \leqq 1 \iff |3k - 2| \leqq \sqrt{k^2 + 1}$$
> 3. 両辺を2乗して整理する。
>    $$(3k - 2)^2 \leqq k^2 + 1 \iff 9k^2 - 12k + 4 \leqq k^2 + 1 \iff 8k^2 - 12k + 3 \leqq 0$$
> 4. 2次方程式 $8k^2 - 12k + 3 = 0$ の解は、解の公式より $k = \frac{3 \pm \sqrt{3}}{4}$。
> 5. 不等式の解は、 $\frac{3 - \sqrt{3}}{4} \leqq k \leqq \frac{3 + \sqrt{3}}{4}$ となる。
> 6. よって、
>    最大値は **$\frac{3 + \sqrt{3}}{4}$**、 最小値は **$\frac{3 - \sqrt{3}}{4}$**。

#### [追加パターンB] 領域の包含関係と円の半径の決定（命題の真偽）
**【鉄則】「命題 $p \implies q$ が真である」 $\iff$ 「領域の包含関係 $P \subset Q$ が成り立つ」！ 包含関係を保つ限界の境界条件（「円が領域の内側に接する条件」や「領域のすべての角を円が含む条件」）を見極めよ！**

> **【例題 (不等式の表す領域PDF P.4 練習7(2)(3)・超重要)】**
> 条件 $p, q$ を次のように定める。
> $p : (x-1)^2 + (y-1)^2 \leqq r^2$  ($r>0$)
> $q : |x| + 2|y| \leqq 4$
> (1) 命題「 $p \implies q$ 」が成り立つような正の数 $r$ の最大値を求めよ。
> (2) 命題「 $q \implies p$ 」が成り立つような正の数 $r$ の最小値を求めよ。
> 
> **【思考プロセスと解答】**
> 条件 $p$ の表す領域 $P$ は、中心 $A(1, 1)$、半径 $r$ の円の内部および周である。
> 条件 $q$ の表す領域 $Q$ は、ひし形（頂点 $(4,0), (0,2), (-4,0), (0,-2)$）の内部および周である。（※パターン26参照）
> 
> **(1) 「 $p \implies q$ 」が成り立つ $\iff P \subset Q$**
> 1. 円 $P$ がひし形 $Q$ の内側に完全に含まれる条件を考える。
> 2. 円の中心 $A(1, 1)$ は第1象限にあり、ひし形 $Q$ の第1象限における境界線は直線 $x + 2y = 4 \iff x + 2y - 4 = 0$ ･･･① である。
> 3. 円 $P$ がひし形 $Q$ からはみ出さない（内包される）ための限界条件は、**中心 $A(1, 1)$ と直線 ① の距離 $d$ が、円の半径 $r$ 以上**であること。
>    $$d = \frac{|1 \cdot 1 + 2 \cdot 1 - 4|}{\sqrt{1^2 + 2^2}} = \frac{|-1|}{\sqrt{5}} = \frac{1}{\sqrt{5}}$$
> 4. したがって、 $r \leqq \frac{1}{\sqrt{5}}$ が条件。
>    よって、求める $r$ の最大値は **$\frac{\sqrt{5}}{5}$** （または $\frac{1}{\sqrt{5}}$）。
> 
> **(2) 「 $q \implies p$ 」が成り立つ $\iff Q \subset P$**
> 1. ひし形 $Q$ が円 $P$ の内側に完全に含まれる条件を考える。
> 2. 多角形が円に内包されるための条件は、**多角形のすべての頂点が円の内部（または周上）にある**ことである。
> 3. 円の中心 $A(1, 1)$ からひし形 $Q$ の4つの頂点までの距離をそれぞれ求める。
>    *   $(4, 0)$ までの距離の2乗： $(4-1)^2 + (0-1)^2 = 10$
>    *   $(0, 2)$ までの距離の2乗： $(0-1)^2 + (2-1)^2 = 2$
>    *   $(-4, 0)$ までの距離の2乗： $(-4-1)^2 + (0-1)^2 = 26$
>    *   $(0, -2)$ までの距離の2乗： $(0-1)^2 + (-2-1)^2 = 10$
> 4. 最も遠い頂点は点 $(-4, 0)$ であり、その距離は $\sqrt{26}$ である。
> 5. ひし形全体を円が覆うためには、円の半径 $r$ がこの最大距離以上でなければならない。
>    $$r \geqq \sqrt{26}$$
> 6. よって、求める $r$ の最小値は **$\sqrt{26}$**。

## 第2部：図形と方程式（軌跡と領域） 第4章
**〜 通過領域（存在条件・値の範囲・境界値問題）の完全攻略 〜**

---

## 1. 通過領域の基本

**① 通過領域とは**
*   直線や放物線などの図形が、含まれるパラメータ（文字定数 $t$ など）の値を変化させながら動くとき、その図形が塗りつぶす（通過する）領域のこと。

**② 通過領域を求める2大アプローチ**
1.  **「逆手流」（実数存在条件）**：
    「点 $(x, y)$ が通過領域内の点である」 $\iff$ 「その点 $(x, y)$ を与えるような実数 $t$ が存在する」。
    直線の式などを **$t$ の方程式と捉え直し**、その方程式が実数解（または指定された範囲内の実数解）をもつ条件（判別式など）を求める。
2.  **「順像法」（変数の1文字固定・値の範囲）**：
    $x$ の値をある定数として**固定**し、パラメータ $t$ を動かしたときに、 $y$ がとりうる値の範囲を特定する。得られた $y$ の範囲（ $x$ の式で表される）が、そのまま通過領域の境界となる。
    *(※特にパラメータ $t$ に範囲制限がある場合は、この「1文字固定法」が極めて安全かつ強力である。)*

---

## 2. 最難関・定番解法パターン

#### [パターン30] 実数全体を動くパラメータによる放物線の通過領域
**【鉄則】直線や放物線の式を $t$ の2次方程式に整理し、「実数 $t$ が存在する $\iff$ 判別式 $D \geqq 0$」に帰着させよ！**

> **【例題 (P.047 例題16、通過領域PDF Q1(1))】**
> 実数 $t$ がすべての実数値を動くとき、放物線 $y = 2tx - t^2$ が通過する領域を図示せよ。
> 
> **【思考プロセスと解答（逆手流）】**
> 1. 「点 $(x, y)$ が求める通過領域に属する」
>    $\iff$ 「等式 $y = 2tx - t^2$ を満たす実数 $t$ が少なくとも1つ存在する」
> 2. 与式を **$t$ の2次方程式**に整理する。
>    $$t^2 - 2xt + y = 0 \quad \cdots\text{①}$$
> 3. ①が実数解をもつ条件は、判別式を $D$ とすると、 $D \geqq 0$ である。
>    $$\frac{D}{4} = (-x)^2 - 1 \cdot y \geqq 0$$
>    $$x^2 - y \geqq 0 \iff y \leqq x^2$$
> 4. よって、求める通過領域は、 **放物線 $y = x^2$ の下側の部分** である。（境界線を含む）

#### [パターン31] 範囲制限を伴う通過領域（1文字固定法）
**【鉄則】パラメータ $t$ に範囲制限（ $0 \leqq t \leqq 1$ など）がある場合は、 $x$ を固定して「$t$ の関数としての $y$ の値域」を求めよ！ 解の配置問題の複雑な場合分けを回避できる！**

> **【例題 (通過領域PDF Q1(2))】**
> 実数 $t$ が $0 \leqq t \leqq 1$ の範囲を動くとき、直線 $y = 2tx - t^2$ が通過する領域を図示せよ。
> 
> **【思考プロセスと解答（1文字固定法）】**
> 1. $x$ の値を1つの定数として**固定**し、 $t$ の関数 $f(t) = -t^2 + 2xt$ （$0 \leqq t \leqq 1$）を考える。
> 2. この関数の最大値・最小値（値域）を求めるため、平方完成して軸の位置（$t = x$）で場合分けを行う。
>    $$f(t) = -(t - x)^2 + x^2$$
>    *   **（i） $x < 0$ のとき**： 軸は変域の左側にある。
>        $f(t)$ は $0 \leqq t \leqq 1$ で単調減少する。
>        最大値は $f(0) = 0$、 最小値は $f(1) = 2x - 1$。
>        よって、 $2x - 1 \leqq y \leqq 0$
>    *   **（ii） $0 \leqq x < \frac{1}{2}$ のとき**： 軸は変域内の左寄りにある。
>        最大値は頂点 $f(x) = x^2$、 最小値は軸から遠い端点 $f(1) = 2x - 1$。
>        よって、 $2x - 1 \leqq y \leqq x^2$
>    *   **（iii） $\frac{1}{2} \leqq x \leqq 1$ のとき**： 軸は変域内の右寄りにある。
>        最大値は頂点 $f(x) = x^2$、 最小値は軸から遠い端点 $f(0) = 0$。
>        よって、 $0 \leqq y \leqq x^2$
>    *   **（iv） $x > 1$ のとき**： 軸は変域の右側にある。
>        $f(t)$ は $0 \leqq t \leqq 1$ で単調増加する。
>        最大値は $f(1) = 2x - 1$、 最小値は $f(0) = 0$。
>        よって、 $0 \leqq y \leqq 2x - 1$
> 3. 各場合分けの $y$ の範囲を座標平面上に図示する。
>    境界線は、放物線 $y = x^2$、直線 $y = 2x - 1$、および $x$ 軸（$y = 0$）である。
>    求める領域は、下図の斜線部分（境界線を含む）。

```
        y ^ 
          |       / y=2x-1
          |  _..-' (1, 1)
          .-'   /
        .'|    /  
   ____/_ |___/________> x
      /  O|  1
     /    |
```

#### [パターン32] 直線が「2点の間を通る」条件（境界値の積）
**【鉄則】「直線 $f(x, y) = ax - y + b = 0$ が2点 $P, Q$ の間を通る」 $\iff$ 「$P$ と $Q$ が直線の反対側にある」 $\iff$ 「代入した値の積が負（ $f(P)f(Q) < 0$ ）」！**

> **【例題 (通過領域ついか PDF Q3)】**
> 直線 $y = ax + b$ は 2点 $P(1, -1), \ Q(2, 1)$ の間を通る。ただし、直線は $P, Q$ を通らないものとする。このとき、点 $(a, b)$ の存在範囲を $ab$ 平面上に図示せよ。
> 
> **【思考プロセスと解答】**
> 1. 直線の式を一般形にする。
>    $$ax - y + b = 0$$
>    これを利用して、関数 $f(x, y) = ax - y + b$ を定義する。
> 2. 「直線が2点 $P(1, -1), Q(2, 1)$ の間を通る」ということは、2点が直線を挟んで反対側に位置するということである。
>    したがって、一方を代入した値が正ならば、他方を代入した値は負になる。
>    $$\iff \mathbf{f(1, -1) \cdot f(2, 1) < 0}$$
> 3. それぞれの座標を代入する。
>    *   $f(1, -1) = a(1) - (-1) + b = a + b + 1$
>    *   $f(2, 1) = a(2) - 1 + b = 2a + b - 1$
> 4. 不等式にする。
>    $$(a + b + 1)(2a + b - 1) < 0$$
> 5. この積の形の不等式を分解する（パターン27の適用）。
>    $$\begin{cases} b > -a - 1 \\ b < -2a + 1 \end{cases} \quad\text{または}\quad \begin{cases} b < -a - 1 \\ b > -2a + 1 \end{cases}$$
> 6. これらを $ab$ 平面（横軸 $a$、縦軸 $b$）に図示する。
>    境界線は、2直線 $b = -a - 1$ と $b = -2a + 1$ である。
>    求める領域は、この2直線の交点 $(2, -3)$ を挟んで上下に広がる、境界線を含まない領域である。
