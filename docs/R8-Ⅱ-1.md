## 1. 加法定理の応用（要点と公式）

### 1.1 2倍角・半角・3倍角の公式
加法定理から派生する、角の大きさを自在に変形するための必須公式である。

**① 2倍角の公式**
*   **正弦**：
    $$\sin 2\theta = 2\sin\theta\cos\theta$$
*   **余弦**：
    $$\cos 2\theta = \cos^2\theta - \sin^2\theta = 1 - 2\sin^2\theta = 2\cos^2\theta - 1$$
    *(※ 状況に応じて、$\sin$ のみ、または $\cos$ のみの式にいつでも統一できるように、3つの変形をすべて暗記すること。)*
*   **正接**：
    $$\tan 2\theta = \frac{2\tan\theta}{1-\tan^2\theta}$$

**② 半角の公式（「次数下げ」の基本）**
$\cos 2\theta$ の公式を平方の形について解くことで得られる。2次式を1次式に下げたいときに威力を発揮する。
*   $$\sin^2\frac{\theta}{2} = \frac{1-\cos\theta}{2}$$
*   $$\cos^2\frac{\theta}{2} = \frac{1+\cos\theta}{2}$$
*   $$\tan^2\frac{\theta}{2} = \frac{1-\cos\theta}{1+\cos\theta}$$

**③ 3倍角の公式**
*   **正弦**：
    $$\sin 3\theta = 3\sin\theta - 4\sin^3\theta \quad \text{（サンシャイン、しんさん）}$$
*   **余弦**：
    $$\cos 3\theta = 4\cos^3\theta - 3\cos\theta \quad \text{（よいこ、よっこさん）}$$

---

### 1.2 点の回転移動の公式（加法定理の幾何応用）
平面上で、原点 $O(0, 0)$ を中心として、点 $P(x_0, y_0)$ を角 $\theta$ だけ回転させた点 $Q(x, y)$ の座標は、加法定理により次のように表される。

$$x = x_0\cos\theta - y_0\sin\theta$$
$$y = x_0\sin\theta + y_0\cos\theta$$

**【導出】**
$OP = r$、動径 $OP$ と $x$ 軸の正の向きとのなす角を $\alpha$ とおくと、 $x_0 = r\cos\alpha, \ y_0 = r\sin\alpha$ である。
点 $Q$ は $OP$ をさらに $\theta$ だけ回転させた点なので、 $Q(r\cos(\alpha+\theta), \ r\sin(\alpha+\theta))$ と表される。加法定理より、
*   $x = r\cos(\alpha+\theta) = r\cos\alpha\cos\theta - r\sin\alpha\sin\theta = x_0\cos\theta - y_0\sin\theta$
*   $y = r\sin(\alpha+\theta) = r\sin\alpha\cos\theta + r\cos\alpha\sin\theta = x_0\sin\theta + y_0\cos\theta$

## 2. 最難関・定番解法パターン

### [パターン1] 2倍角・半角公式を用いた方程式・不等式（角の統一）
**【鉄則】$\sin 2\theta$ と $\cos \theta$（または $\sin \theta$）が混在する方程式・不等式は、2倍角の公式を用いて「角度を $\theta$ に統一」し、因数分解の形 $( \quad )( \quad ) = 0$ に持ち込め！**

> **【例題 (問題集P.121 294(1)類題)】**
> $0 \leqq \theta < 2\pi$ のとき、方程式 $\cos 2\theta + \sin\theta = 0$ を解け。
> 
> **【思考プロセスと解答】**
> 1. $\cos 2\theta$ と $\sin\theta$ で角の大きさが異なる。 $\cos 2\theta = 1 - 2\sin^2\theta$ を用いて、角を $\theta$ に、関数を $\sin\theta$ に統一する。
>    $$(1 - 2\sin^2\theta) + \sin\theta = 0$$
> 2. 式を整理する。
>    $$2\sin^2\theta - \sin\theta - 1 = 0$$
> 3. たすき掛けを用いて因数分解する。
>    $$(2\sin\theta + 1)(\sin\theta - 1) = 0$$
>    よって、 $\sin\theta = -\frac{1}{2}, \ 1$。
> 4. $0 \leqq \theta < 2\pi$ の範囲で、これを満たす $\theta$ を単位円から求める。
>    *   $\sin\theta = -\frac{1}{2}$ のとき、 $\theta = \frac{7}{6}\pi, \ \frac{11}{6}\pi$
>    *   $\sin\theta = 1$ のとき、 $\theta = \frac{\pi}{2}$
> 5. したがって、求める解は、 $\mathbf{\theta = \frac{\pi}{2}, \ \frac{7}{6}\pi, \ \frac{11}{6}\pi}$。

### [パターン2] 3倍角の公式の利用（等式・幾何への応用）
**【鉄則】3次式が含まれる三角関数の等式は、3倍角の公式を用いて次数を下げるか、または $\theta = 72^\circ$ のような幾何的な関係式から因数定理に持ち込め！**

> **【例題 (問題集P.122 156参考)】**
> 半径 $1$ の円に内接する正五角形 $ABCDE$ の1つの内角の性質から、等式 $\sin 3\theta + \sin 2\theta = 0$ （$\theta = \frac{2}{5}\pi$）が成り立つ。これを利用して、 $\cos\frac{2}{5}\pi$ の値を求めよ。
> 
> **【思考プロセスと解答】**
> 1. $\theta = \frac{2}{5}\pi$ より、 $5\theta = 2\pi \iff 3\theta = 2\pi - 2\theta$。
>    両辺の $\sin$ をとると、 $\sin 3\theta = \sin(2\pi - 2\theta) = -\sin 2\theta$。
>    よって、 $\sin 3\theta + \sin 2\theta = 0$ が成り立つ。
> 2. この式を、倍角・3倍角の公式を用いて $\sin\theta, \cos\theta$ の等式に変形する。
>    $$(3\sin\theta - 4\sin^3\theta) + 2\sin\theta\cos\theta = 0$$
> 3. $\theta = \frac{2}{5}\pi$ より $\sin\theta \neq 0$ なので、両辺を $\sin\theta$ で割る。
>    $$3 - 4\sin^2\theta + 2\cos\theta = 0$$
> 4. $\sin^2\theta = 1 - \cos^2\theta$ を代入し、 $\cos\theta$ だけの式にする。
>    $$3 - 4(1 - \cos^2\theta) + 2\cos\theta = 0 \iff 4\cos^2\theta + 2\cos\theta - 1 = 0$$
> 5. $\cos\theta = t$ とおくと、 $4t^2 + 2t - 1 = 0$。 解の公式より、
>    $$t = \frac{-1 \pm \sqrt{1 - 4 \cdot 1 \cdot (-1)}}{4} = \frac{-1 \pm \sqrt{5}}{4}$$
> 6. $\theta = \frac{2}{5}\pi = 72^\circ$ は第1象限の角なので、 $\cos\theta > 0$ である。
>    したがって、求める値は $\mathbf{\cos\frac{2}{5}\pi = \frac{-1 + \sqrt{5}}{4}}$。

### [パターン3] 加法定理を用いた点の回転移動
**【鉄則】原点 $O$ を中心とする回転移動は、 「$x' = x_0\cos\theta - y_0\sin\theta$」「$y' = x_0\sin\theta + y_0\cos\theta$」 の公式に代入して処理せよ！**

> **【例題 (問題集レベル演習第1回 1類題)】**
> 点 $P(-1, 2)$ を、原点 $O$ を中心として $\frac{\pi}{6}$ だけ回転させた点 $Q$ の座標を求めよ。
> 
> **【思考プロセスと解答】**
> 1. 回転の公式において、 $x_0 = -1, \ y_0 = 2, \ \theta = \frac{\pi}{6}$ を代入する。
> 2. $\cos\frac{\pi}{6} = \frac{\sqrt{3}}{2}, \ \sin\frac{\pi}{6} = \frac{1}{2}$ である。
> 3. 各座標を計算する。
>    $$x = x_0\cos\theta - y_0\sin\theta = (-1) \cdot \frac{\sqrt{3}}{2} - 2 \cdot \frac{1}{2} = -\frac{\sqrt{3}}{2} - 1 = \frac{-\sqrt{3}-2}{2}$$
>    $$y = x_0\sin\theta + y_0\cos\theta = (-1) \cdot \frac{1}{2} + 2 \cdot \frac{\sqrt{3}}{2} = -\frac{1}{2} + \sqrt{3} = \frac{2\sqrt{3}-1}{2}$$
> 4. よって、求める点 $Q$ の座標は **$\mathbf{\left( \frac{-\sqrt{3}-2}{2}, \ \frac{2\sqrt{3}-1}{2} \right)}$**。

### [パターン4] 三角不等式による円周率 $\pi$ の評価
**【鉄則】鋭角 $\theta$ （$0 < \theta < \frac{\pi}{2}$）において常に成り立つ不等式 $\mathbf{\sin\theta < \theta < \tan\theta}$ に、半角の公式が使える $\theta = \frac{\pi}{12} \ (=15^\circ)$ などの特殊角を代入し、各辺を定数倍して $\pi$ を挟み込め！**

> **【例題 (問題集P.253 重要例題157)】**
> 不等式 $\sin\theta < \theta < \tan\theta$ （$0 < \theta < \frac{\pi}{2}$）を用いて、次の不等式が成り立つことを証明せよ。
> $$3\sqrt{6} - 3\sqrt{2} < \pi < 24 - 12\sqrt{3}$$
> 
> **【思考プロセスと解答】**
> 1. 不等式に $\theta = \frac{\pi}{12}$ を代入する（※ $0 < \frac{\pi}{12} < \frac{\pi}{2}$ を満たしている）。
>    $$\sin\frac{\pi}{12} < \frac{\pi}{12} < \tan\frac{\pi}{12}$$
> 2. 各辺に $12$ を掛ける。
>    $$12\sin\frac{\pi}{12} < \pi < 12\tan\frac{\pi}{12} \quad \cdots\text{①}$$
> 3. 半角の公式を用いて、各辺の三角比の値を求める。
>    *   $\sin^2\frac{\pi}{12} = \frac{1-\cos\frac{\pi}{6}}{2} = \frac{1-\frac{\sqrt{3}}{2}}{2} = \frac{2-\sqrt{3}}{4} = \frac{8-4\sqrt{3}}{8} = \frac{4-2\sqrt{3}}{8} \implies \sin\frac{\pi}{12} = \frac{\sqrt{6}-\sqrt{2}}{4}$
>    *   $\tan^2\frac{\pi}{12} = \frac{1-\cos\frac{\pi}{6}}{1+\cos\frac{\pi}{6}} = \frac{2-\sqrt{3}}{2+\sqrt{3}} = (2-\sqrt{3})^2 \implies \tan\frac{\pi}{12} = 2-\sqrt{3}$
> 4. 得られた値を ① に代入する。
>    *   左辺 $= 12 \left( \frac{\sqrt{6}-\sqrt{2}}{4} \right) = 3\sqrt{6} - 3\sqrt{2}$
>    *   右辺 $= 12(2-\sqrt{3}) = 24 - 12\sqrt{3}$
> 5. したがって、 $3\sqrt{6} - 3\sqrt{2} < \pi < 24 - 12\sqrt{3}$ が成り立つ。 （証明終）

## 1. 合成と対称式（要点と公式）

**① 三角関数の合成公式**
同周期の正弦（$\sin$）と余弦（$\cos$）の1次式の和を、1つの正弦の式に統合する極めて強力な技術である。
$$\mathbf{a\sin\theta + b\cos\theta = \sqrt{a^2+b^2} \sin(\theta+\alpha)}$$
ただし、 $\cos\alpha = \frac{a}{\sqrt{a^2+b^2}}, \quad \sin\alpha = \frac{b}{\sqrt{a^2+b^2}}$
*   **【図形的意味】** 座標平面上に点 $P(a, b)$ をとったとき、線分 $OP$ の長さが新しい振幅 $\sqrt{a^2+b^2}$ となり、 $OP$ が $x$ 軸の正の向きとなす角が平行移動量 $\alpha$ となる。

**② 対称式 $t = \sin\theta + \cos\theta$ のおきかえの背景**
*   $\sin\theta$ と $\cos\theta$ の対称式は、 $t = \sin\theta + \cos\theta$ と置くことで $t$ の多項式に変換できる。
*   両辺を2乗すると、 $\sin^2\theta + 2\sin\theta\cos\theta + \cos^2\theta = t^2 \iff 1 + \sin 2\theta = t^2$ より、
    $$\sin\theta\cos\theta = \frac{t^2-1}{2}$$
    として積の項を $t$ で表すことができる。
*   **【注意】** 置き換えた変数 $t$ の変域は、上記の合成公式を用いて必ず制限しなければならない。
    $$t = \sqrt{2}\sin\left(\theta+\frac{\pi}{4}\right) \implies \mathbf{-\sqrt{2} \leqq t \leqq \sqrt{2}} \quad (\text{※ } 0 \leqq \theta < 2\pi \text{ のとき})$$

## 2. 最難関・定番解法パターン

### [パターン5] 三角関数の合成を用いた方程式・不等式の解法
**【鉄則】$\sin\theta$ と $\cos\theta$ の1次式の和を含む方程式・不等式は、合成を用いて1つの $\sin$ に統合し、角の範囲を拡張して単位円から解を求めよ！**

> **【例題 (問題集P.260 基本例題161類題)】**
> $0 \leqq \theta < 2\pi$ のとき、方程式 $\sqrt{3}\sin\theta + \cos\theta = 1$ を解け。
> 
> **【思考プロセスと解答】**
> 1. 左辺を合成する。点 $(\sqrt{3}, 1)$ をとると、原点との距離は $2$、なす角は $\frac{\pi}{6}$。
>    $$2\sin\left(\theta + \frac{\pi}{6}\right) = 1 \iff \sin\left(\theta + \frac{\pi}{6}\right) = \frac{1}{2}$$
> 2. $\theta + \frac{\pi}{6} = x$ とおき、新変域を求める。
>    $$0 \leqq \theta < 2\pi \implies \mathbf{\frac{\pi}{6} \leqq \theta + \frac{\pi}{6} < \frac{13\pi}{6}}$$
> 3. この範囲で $\sin x = \frac{1}{2}$ を満たす $x$ を単位円上で探す。
>    $$x = \frac{\pi}{6}, \ \frac{5}{6}\pi$$
> 4. $\theta$ に戻して解を求める。
>    *   $\theta + \frac{\pi}{6} = \frac{\pi}{6} \implies \theta = 0$
>    *   $\theta + \frac{\pi}{6} = \frac{5}{6}\pi \implies \theta = \frac{2}{3}\pi$
> 5. よって、求める解は $\mathbf{\theta = 0, \ \frac{2}{3}\pi}$。

### [パターン6] 同次2次式 $a\sin^2\theta+b\sin\theta\cos\theta+c\cos^2\theta$ の最大・最小
**【鉄則】$\sin^2\theta, \ \cos^2\theta, \ \sin\theta\cos\theta$ の2次の同次式は、半角の公式と2倍角の公式を用いて「角 $2\theta$ の1次式」に次数を下げ、最後に「合成」で仕留めよ！**

> **【例題 (問題集P.264 基本例題164)】**
> 関数 $y = \sin^2\theta + 2\sin\theta\cos\theta - 3\cos^2\theta$ の最大値、最小値を求めよ。
> 
> **【思考プロセスと解答】**
> 1. 半角の公式、2倍角の公式を用いて、各項を角 $2\theta$ の1次式に直す。
>    *   $\sin^2\theta = \frac{1-\cos 2\theta}{2}$
>    *   $2\sin\theta\cos\theta = \sin 2\theta$
>    *   $\cos^2\theta = \frac{1+\cos 2\theta}{2}$
> 2. これらを与式に代入して整理する。
>    $$y = \frac{1-\cos 2\theta}{2} + \sin 2\theta - 3 \cdot \frac{1+\cos 2\theta}{2}$$
>    $$= \sin 2\theta - 2\cos 2\theta - 1$$
> 3. 三角関数の合成を実行する。点 $(1, -2)$ をとると、原点との距離は $\sqrt{1^2+(-2)^2} = \sqrt{5}$。
>    $$y = \sqrt{5}\sin(2\theta + \alpha) - 1$$
>    （ただし、 $\alpha$ は $\cos\alpha = \frac{1}{\sqrt{5}}, \ \sin\alpha = -\frac{2}{\sqrt{5}}$ を満たす角）
> 4. $\theta$ に制限がないため、 $-1 \leqq \sin(2\theta + \alpha) \leqq 1$ である。
>    *   最大値： $\sqrt{5}(1) - 1 = \mathbf{\sqrt{5} - 1}$
>    *   最小値： $\sqrt{5}(-1) - 1 = \mathbf{-\sqrt{5} - 1}$

### [パターン7] 対称式型（$t = \sin\theta + \cos\theta$）の最大・最小、方程式
**【鉄則】$\sin\theta + \cos\theta = t$ とおいたときは、必ず合成公式を用いて $t$ の変域制限（$-\sqrt{2} \leqq t \leqq \sqrt{2}$ など）を横に明記し、2次関数の最大最小に帰着させよ！**

> **【例題 (問題集P.262 基本例題163)】**
> 関数 $y = \sin 2\theta + \sin\theta + \cos\theta$ の最大値、最小値を求めよ。（ただし $0 \leqq \theta < 2\pi$）
> 
> **【思考プロセスと解答】**
> 1. $\sin\theta + \cos\theta = t$ とおく。
>    合成公式より、 $t = \sqrt{2}\sin\left(\theta + \frac{\pi}{4}\right)$ である。
>    $0 \leqq \theta < 2\pi$ より、 $t$ のとりうる値の範囲は **$-\sqrt{2} \leqq t \leqq \sqrt{2}$** ･･･①
> 2. 両辺を2乗して、積の項を $t$ の式にする。
>    $t^2 = 1 + \sin 2\theta \implies \sin 2\theta = t^2 - 1$
> 3. これらを関数に代入し、 $t$ の2次関数に書き換える。
>    $$y = (t^2 - 1) + t = t^2 + t - 1$$
> 4. ①の範囲で平方完成して、頂点と端点の値を比較する。
>    $$y = \left(t + \frac{1}{2}\right)^2 - \frac{5}{4}$$
>    *   軸は $t = -\frac{1}{2}$ であり、①の変域内にある。 よって、 $t = -\frac{1}{2}$ のとき、最小値は $\mathbf{-\frac{5}{4}}$。
>    *   軸から最も遠い端点 $t = \sqrt{2}$ のとき、最大値は $(\sqrt{2})^2 + \sqrt{2} - 1 = \mathbf{1 + \sqrt{2}}$。

### [パターン8] 【最難関・幾何融合】三角形の角度を用いた最大・最小
**【鉄則】三角形の内角の和 $A+B+C = \pi$ や、正弦定理・余弦定理を用いて、求めたい幾何的な量を「1つの角度 $B$ のみの三角関数」に表し、最後に合成を用いて最大値を求めよ！**

> **【例題 (問題集P.268 基本例題167)】**
> $\triangle ABC$ の外接円の半径が $1$ である。 $\angle A = \frac{\pi}{3}$ のとき、3辺の長さの和 $a+b+c$ の最大値を求めよ。
> 
> **【思考プロセスと解答】**
> 1. 正弦定理 $\frac{a}{\sin A} = \frac{b}{\sin B} = \frac{c}{\sin C} = 2R$ を利用する。
>    $R = 1$ より、 $a = 2\sin A, \ b = 2\sin B, \ c = 2\sin C$ である。
> 2. $\angle A = \frac{\pi}{3}$ より、 $a = 2\sin\frac{\pi}{3} = 2 \cdot \frac{\sqrt{3}}{2} = \sqrt{3}$。
> 3. 三角形の内角の和の関係式 $A+B+C = \pi$ より、
>    $$B+C = \pi - \frac{\pi}{3} = \frac{2}{3}\pi \implies C = \frac{2}{3}\pi - B$$
>    また、 $B > 0, \ C > 0$ より、角 $B$ のとりうる値の範囲は **$0 < B < \frac{2}{3}\pi$** ･･･①
> 4. 3辺の和 $a+b+c$ を、変数 $B$ の式で表す。
>    $$a+b+c = \sqrt{3} + 2\sin B + 2\sin\left(\frac{2}{3}\pi - B\right)$$
> 5. 加法定理を用いて展開し、整理する。
>    $$\sin\left(\frac{2}{3}\pi - B\right) = \sin\frac{2}{3}\pi\cos B - \cos\frac{2}{3}\pi\sin B = \frac{\sqrt{3}}{2}\cos B + \frac{1}{2}\sin B$$
>    よって、
>    $$2\sin B + 2\sin\left(\frac{2}{3}\pi - B\right) = 2\sin B + \sqrt{3}\cos B + \sin B = 3\sin B + \sqrt{3}\cos B$$
> 6. 三角関数の合成を実行する。
>    $$3\sin B + \sqrt{3}\cos B = \sqrt{3^2 + (\sqrt{3})^2} \sin(B + \alpha) = 2\sqrt{3}\sin\left(B + \frac{\pi}{6}\right)$$
> 7. 3辺の和は、 $a+b+c = \sqrt{3} + 2\sqrt{3}\sin\left(B + \frac{\pi}{6}\right)$。
>    ①より $\frac{\pi}{6} < B + \frac{\pi}{6} < \frac{5}{6}\pi$ であるから、
>    $\sin\left(B + \frac{\pi}{6}\right)$ は $B + \frac{\pi}{6} = \frac{\pi}{2} \iff B = \frac{\pi}{3}$ （このとき $C = \frac{\pi}{3}$ となり、正三角形になる）のとき、最大値 $1$ をとる。
> 8. したがって、求める最大値は $\sqrt{3} + 2\sqrt{3}(1) = \mathbf{3\sqrt{3}}$。

### [追加パターンB] 条件付き2変数2次式の最大・最小
**【鉄則】「条件 $x^2+y^2=1$」を満たす実数 $x, y$ の2次式の最大・最小は、 $\mathbf{x = \cos\theta, \ y = \sin\theta}$ とパラメータ表示せよ！ これにより、[パターン6] と全く同じ1変数三角関数の次数下げ・合成問題に100%帰着できる！**

> **【例題 (問題集P.265 重要例題165)】**
> 実数 $x, y$ が $x^2 + y^2 = 1$ を満たすとき、 $3x^2 + 2xy + y^2$ の最大値と最小値を求めよ。
> 
> **【思考プロセスと解答】**
> 1. $x^2 + y^2 = 1$ より、 $x = \cos\theta, \ y = \sin\theta$ （$0 \leqq \theta < 2\pi$）とおく。
> 2. 与えられた2次式に代入する。
>    $$3x^2 + 2xy + y^2 = 3\cos^2\theta + 2\sin\theta\cos\theta + \sin^2\theta$$
> 3. [パターン6] と同様に、半角・倍角の公式を用いて次数を下げる。
>    $$= 3 \cdot \frac{1+\cos 2\theta}{2} + \sin 2\theta + \frac{1-\cos 2\theta}{2}$$
>    $$= \sin 2\theta + \cos 2\theta + 2$$
> 4. 三角関数の合成を実行する。
>    $$\sin 2\theta + \cos 2\theta = \sqrt{2}\sin\left(2\theta + \frac{\pi}{4}\right)$$
>    よって、式は $\sqrt{2}\sin\left(2\theta + \frac{\pi}{4}\right) + 2$ となる。
> 5. $0 \leqq \theta < 2\pi$ において、 $-1 \leqq \sin\left(2\theta + \frac{\pi}{4}\right) \leqq 1$ であるから、
>    *   最大値は $\sqrt{2}(1) + 2 = \mathbf{2 + \sqrt{2}}$
>    *   最小値は $\sqrt{2}(-1) + 2 = \mathbf{2 - \sqrt{2}}$

#### [追加パターンC] 2乗して加える加法定理の応用（三角関数の連立式）

> 
> **【思考プロセスと解答】**
> 1. ①と②の両辺をそれぞれ2乗する。
>    *   ①の2乗： $\sin^2\alpha + 2\sin\alpha\sin\beta + \sin^2\beta = 1$ ･･･③
>    *   ②の2乗： $\cos^2\alpha - 2\cos\alpha\cos\beta + \cos^2\beta = \frac{1}{4}$ ･･･④
> 2. ③と④を辺々足し合わせる。
>    $$(\sin^2\alpha + \cos^2\alpha) + (\sin^2\beta + \cos^2\beta) - 2(\cos\alpha\cos\beta - \sin\alpha\sin\beta) = 1 + \frac{1}{4}$$
> 3. 相互関係 $\sin^2\theta + \cos^2\theta = 1$ および、加法定理 $\cos\alpha\cos\beta - \sin\alpha\sin\beta = \cos(\alpha+\beta)$ を適用して整理する。
>    $$1 + 1 - 2\cos(\alpha+\beta) = \frac{5}{4}$$

> 4. 両辺を $2$ で割る。
>    $$\cos(\alpha+\beta) = \mathbf{\frac{3}{8}}$$

## 3. 積和・和積の公式（要点と公式）

加法定理から導かれる公式群であり、項の「掛け算」を「足し算・引き算」に、あるいは「足し算・引き算」を「掛け算」に変換できる。これにより、それまで因数分解や簡略化が困難だった三角関数の計算・方程式が劇的に解きやすくなる。

**① 積を和・差に直す公式（積和公式）**
加法定理の正弦・余弦の式を辺々足し引きすることで導かれる。
*   $$\sin\alpha\cos\beta = \frac{1}{2}\{\sin(\alpha+\beta) + \sin(\alpha-\beta)\}$$
*   $$\cos\alpha\sin\beta = \frac{1}{2}\{\sin(\alpha+\beta) - \sin(\alpha-\beta)\}$$
*   $$\cos\alpha\cos\beta = \frac{1}{2}\{\cos(\alpha+\beta) + \cos(\alpha-\beta)\}$$
*   $$\sin\alpha\sin\beta = -\frac{1}{2}\{\cos(\alpha+\beta) - \cos(\alpha-\beta)\}$$

**② 和・差を積に直す公式（和積公式）**
積和公式において、 $\alpha+\beta = A, \ \alpha-\beta = B$ と置き換えることで導かれる。
*   $$\sin A + \sin B = 2\sin\frac{A+B}{2}\cos\frac{A-B}{2}$$
*   $$\sin A - \sin B = 2\cos\frac{A+B}{2}\sin\frac{A-B}{2}$$
*   $$\cos A + \cos B = 2\cos\frac{A+B}{2}\cos\frac{A-B}{2}$$
*   $$\cos A - \cos B = -2\sin\frac{A+B}{2}\sin\frac{A-B}{2}$$

### [パターン9] 積和公式を用いた角の消去（$\cos 20^\circ \cos 40^\circ \cos 80^\circ$ 型）
**【鉄則】有名角でない三角関数の掛け算は、積和公式を繰り返し適用し、打ち消し合う関係（ $\cos(180^\circ-\theta) = -\cos\theta$ など）や、有名角（ $60^\circ$ など）をあぶり出して値を求めよ！**

> **【例題 (問題集レベル演習第1回 4)】**
> 次の式の値を求めよ。
> $$\cos 20^\circ \cos 40^\circ \cos 80^\circ$$
> 
> **【思考プロセスと解答】**
> 1. まず、前の2項 $\cos 20^\circ \cos 40^\circ$ に積和公式を適用する。
>    $$\cos 20^\circ \cos 40^\circ = \frac{1}{2}(\cos 60^\circ + \cos(-20^\circ)) = \frac{1}{2}\left(\frac{1}{2} + \cos 20^\circ\right)$$
> 2. これを元の式に戻し、展開する。
>    $$(\text{与式}) = \frac{1}{2}\left(\frac{1}{2} + \cos 20^\circ\right) \cos 80^\circ$$
>    $$= \frac{1}{4}\cos 80^\circ + \frac{1}{2}\cos 20^\circ \cos 80^\circ$$
> 3. 後ろの項 $\cos 20^\circ \cos 80^\circ$ に再び積和公式を適用する。
>    $$\cos 20^\circ \cos 80^\circ = \frac{1}{2}(\cos 100^\circ + \cos(-60^\circ)) = \frac{1}{2}\left(\cos 100^\circ + \frac{1}{2}\right)$$
> 4. 代入して整理する。
>    $$(\text{与式}) = \frac{1}{4}\cos 80^\circ + \frac{1}{2} \cdot \frac{1}{2}\left(\cos 100^\circ + \frac{1}{2}\right)$$
>    $$= \frac{1}{4}\cos 80^\circ + \frac{1}{4}\cos 100^\circ + \frac{1}{8}$$
> 5. ここで、角度の関係に着目する。 $100^\circ = 180^\circ - 80^\circ$ であるから、
>    $$\cos 100^\circ = \cos(180^\circ - 80^\circ) = -\cos 80^\circ$$
> 6. これを代入すると、 $\cos 80^\circ$ の項が打ち消し合って消去される。
>    $$(\text{与式}) = \frac{1}{4}\cos 80^\circ - \frac{1}{4}\cos 80^\circ + \frac{1}{8} = \mathbf{\frac{1}{8}}$$

### [パターン10] 和積公式を用いた複数項の三角方程式（因数分解）

> **【例題 (問題集レベル演習第1回 5)】**

>    $$\sin 2x (2\cos x - 1) = 0$$

>        $0 \leqq x < 2\pi$ より、 $2x$ の範囲は $0 \leqq 2x < 4\pi$ （※2周分）である。
>        $$2x = 0, \ \pi, \ 2\pi, \ 3\pi \implies x = 0, \ \frac{\pi}{2}, \ \pi, \ \frac{3}{2}\pi$$
>    *   **（ii） $\cos x = \frac{1}{2}$ のとき**
>        $0 \leqq x < 2\pi$ の範囲において、
>        $$x = \frac{\pi}{3}, \ \frac{5}{3}\pi$$
> 5. 最後にすべての解をまとめて、昇順に並べる。
>    よって、求める解は $\mathbf{x = 0, \ \frac{\pi}{3}, \ \frac{\pi}{2}, \ \pi, \ \frac{3}{2}\pi, \ \frac{5}{3}\pi}$。

## 1. 指数の拡張と指数関数（要点と公式）

### 1.1 指数の拡張と指数法則
指数を「正の整数」から「有理数」まで拡張することで、すべての累乗・累乗根を統一的に計算できるようになる。

**① 指数の拡張の定義**（$a \neq 0$、 $n$ は正の整数とする）
*   **0の指数**：
    $$a^0 = 1$$
*   **負の整数の指数**：
    $$a^{-n} = \frac{1}{a^n}$$
*   **分数指数**（$a > 0$、 $m$ は整数、 $n$ は2以上の整数とする）：
    $$a^{\frac{m}{n}} = \sqrt[n]{a^m} \quad \left(\text{とくに } a^{\frac{1}{n}} = \sqrt[n]{a}\right)$$

**② 累乗根の性質**（$a > 0, \ b > 0$、 $m, n$ は2以上の整数とする）
1.  $$\sqrt[n]{a}\sqrt[n]{b} = \sqrt[n]{ab}$$
2.  $$\frac{\sqrt[n]{a}}{\sqrt[n]{b}} = \sqrt[n]{\frac{a}{b}}$$
3.  $$\left(\sqrt[n]{a}\right)^m = \sqrt[n]{a^m}$$
4.  $$\sqrt[m]{\sqrt[n]{a}} = \sqrt[mn]{a}$$

**③ 指数法則**（$a > 0, \ b > 0$、 $p, q$ は有理数とする）
*   $$a^p a^q = a^{p+q}$$
*   $$\frac{a^p}{a^q} = a^{p-q}$$
*   $$(a^p)^q = a^{pq}$$
*   $$(ab)^p = a^p b^p$$

---

### 1.2 指数関数のグラフと性質
*   **定義**： $a > 0, \ a \neq 1$ とするとき、関数 $\mathbf{y = a^x}$ を $a$ を底とする指数関数という。
*   **特徴**：
    1.  定点 $\mathbf{(0, 1)}$ および $\mathbf{(1, a)}$ を必ず通る。
    2.  $x$ 軸（直線 $y=0$）が**漸近線**となる。
    3.  底 $a$ の範囲によるグラフの挙動：
        *   **$a > 1$ のとき**： **単調増加関数**（ $p < q \iff a^p < a^q$ ）
        *   **$0 < a < 1$ のとき**： **単調減少関数**（ $p < q \iff a^p > a^q$ ）

## 1. 指数方程式・不等式の基本要綱

**① 指数方程式・不等式の基本解法**
底をそろえて、指数の大小を比較する。
*   **方程式**：
    $$a^x = a^y \iff x = y \quad (a > 0, \ a \neq 1)$$
*   **不等式**：
    *   底が $1$ より大きい（ $a > 1$ ）とき：
        $$a^x > a^y \iff x > y \quad (\text{不等号の向きは同じ})$$
    *   底が $1$ より小さい（ $0 < a < 1$ ）とき：
        $$a^x > a^y \iff x < y \quad (\text{不等号の向きは逆})$$

**② $a^x + a^{-x} = t$ の置き換え**
*   $a^x > 0$ かつ $a^{-x} > 0$ であるから、**相加平均と相乗平均の関係**により、
    $$a^x + a^{-x} \geqq 2\sqrt{a^x \cdot a^{-x}} = 2\sqrt{1} = 2$$
    が常に成り立つ。
*   したがって、 $a^x + a^{-x} = t$ と置き換えた場合、変数 $t$ には自動的に $\mathbf{t \geqq 2}$ **という範囲制限（等号成立は $x=0$ のとき）**が発生する。

## 2. 最難関・定番解法パターン

### [パターン14] $t = a^x$ の置き換えによる2次不等式
**【鉄則】 $t = a^x$ と置き換えたら、必ず横に「 $\mathbf{t > 0}$ 」と明記せよ！ 不等式を解いて得られた $t$ の範囲と $t>0$ の共通部分をとることを忘れるな！**

> **【例題 (P.280 基本例題174(1)類題)】**
> 不等式 $9^x - 4 \cdot 3^x + 3 \leqq 0$ を解け。
> 
> **【思考プロセスと解答】**
> 1. $3^x = t$ とおく。指数関数の性質より、 **$t > 0$** ･･･① である。
> 2. 与式を $t$ の不等式に書き換える（ $9^x = (3^2)^x = (3^x)^2 = t^2$ ）。
>    $$t^2 - 4t + 3 \leqq 0$$
> 3. 左辺を因数分解して解く。
>    $$(t-1)(t-3) \leqq 0 \implies 1 \leqq t \leqq 3 \quad \cdots\text{②}$$
> 4. ①と②の共通部分をとると、 $1 \leqq t \leqq 3$（すべて $t>0$ を満たしている）。
> 5. $t = 3^x$ に戻して $x$ を求める。
>    $$3^0 \leqq 3^x \leqq 3^1$$
> 6. 底 $3$ は $1$ より大きい（ $3 > 1$ ）ので、指数の大小関係はそのまま。
>    よって、求める解は $\mathbf{0 \leqq x \leqq 1}$。

### [パターン15] $a^x + a^{-x} = t$ の置き換えによる最大・最小
**【鉄則】 $a^x + a^{-x} = t$ とおいたときは、相加平均・相乗平均の関係から $t \geqq 2$ となる。 $y$ を $t$ の2次関数に直して、軸の位置に注意して最大・最小を求めよ！**

> **【例題 (P.072 例題23)】**
> 関数 $y = 4^x + 4^{-x} - 5(2^x + 2^{-x}) + 9$ の最小値と、そのときの $x$ の値を求めよ。
> 
> **【思考プロセスと解答】**
> 1. $2^x + 2^{-x} = t$ とおく。
>    $2^x > 0, \ 2^{-x} > 0$ であるから、相加平均・相乗平均の関係より、
>    $$t \geqq 2\sqrt{2^x \cdot 2^{-x}} = 2 \quad \cdots\text{①}$$
> 2. $4^x + 4^{-x}$ を $t$ で表す。両辺を2乗すると、
>    $$t^2 = (2^x + 2^{-x})^2 = (2^x)^2 + 2 \cdot 2^x \cdot 2^{-x} + (2^{-x})^2 = 4^x + 2 + 4^{-x}$$
>    よって、 $4^x + 4^{-x} = t^2 - 2$。
> 3. これらを関数に代入して $t$ の式にする。
>    $$y = (t^2 - 2) - 5t + 9 = t^2 - 5t + 7$$
> 4. ①の範囲（ $t \geqq 2$ ）で平方完成し、最少値を求める。
>    $$y = \left(t - \frac{5}{2}\right)^2 + \frac{3}{4}$$
>    軸は $t = \frac{5}{2}$ であり、①の変域に含まれる。
>    よって、 $t = \frac{5}{2}$ のとき、最小値は $\mathbf{\frac{3}{4}}$。
> 5. このときの $x$ の値を求める。
>    $$2^x + 2^{-x} = \frac{5}{2}$$
>    両辺に $2^x$ を掛け、 $2^x = X \ (>0)$ とおいて整理する。
>    $$X + \frac{1}{X} = \frac{5}{2} \iff 2X^2 - 5X + 2 = 0 \iff (2X-1)(X-2) = 0$$
>    よって $X = \frac{1}{2}, \ 2 \implies 2^x = 2^{-1}, \ 2^1 \implies x = \pm 1$。
> 6. 以上より、**$x = \pm 1$ のとき、最小値** $\mathbf{\frac{3}{4}}$。

### [パターン16] 【最難関】指数方程式の実数解の個数（定数分離）
**【鉄則】 $t = a^x \ (>0)$ と置き換えたとき、「異なる実数解 $x$ の個数」は、「$t > 0$ の範囲における実数解 $t$ の個数」と $1$ 対 $1$ に対応する！ 複雑なグラフの場合は、定数を右辺に分離し、 $t > 0$ における共有点の個数として視覚的に解け！**

> **【例題 (問題集レベル演習第7回 1類題)】**
> $a$ を定数とする。方程式 $4^x - 2^{x+2} + 3a + 2 = 0$ ･･･① について、
> (1) ①が異なる2つの実数解をもつような $a$ の値の範囲を求めよ。
> (2) ①が異なる2つの正の実数解をもつような $a$ の値の範囲を求めよ。
> (3) ①が正の解と負の解をそれぞれ1つずつもつような $a$ の値の範囲を求めよ。
> 
> **【思考プロセスと解答】**
> 1. $2^x = t$ とおく。実数 $x$ に対し、 **$t > 0$** ･･･② が成り立つ。
>    このとき、実数解 $x$ の1つの値に対し、正の実数解 $t$ の値がただ1つ対応する。
> 2. 与式を $t$ の方程式にする。 $4^x = t^2$ 、 $2^{x+2} = 4t$ より、
>    $$t^2 - 4t + 3a + 2 = 0 \iff t^2 - 4t + 2 = -3a$$
> 3. 定数 $-3a$ を右辺に分離し、左辺を $g(t) = t^2 - 4t + 2 = (t-2)^2 - 2$ とする。
>    境界値となる $t=0, 1$ のときの値も求めておく。
>    $$g(0) = 2, \quad g(1) = 1^2 - 4(1) + 2 = -1, \quad \text{頂点：} (2, -2)$$
>    これらをもとに、 $t > 0$ における $y = g(t)$ のグラフを描く。
> 
> (1) ①が異なる2つの実数解をもつ
>     $\iff$ 不等式 $g(t) = -3a$ が **$t > 0$ の範囲に異なる2つの実数解**をもつ。
>     グラフより、水平な直線 $Y = -3a$ が、放物線 $y = g(t)$ の $t > 0$ の部分と2つの共有点をもてばよい。
>     頂点の $y$ 座標 $-2$ から、 $t=0$ のときの $y$ 座標 $2$ の間にあれば、2点で交わる。
>     $$-2 < -3a < 2 \iff \mathbf{-\frac{2}{3} < a < \frac{2}{3}}$$
> 
> (2) ①が異なる2つの正の実数解をもつ
>     $x > 0 \iff 2^x > 2^0 \iff t > 1$ である。
>     $\iff$ 不等式 $g(t) = -3a$ が **$t > 1$ の範囲に異なる2つの実数解**をもつ。
>     頂点の $y$ 座標 $-2$ から、 $t=1$ のときの $y$ 座標 $-1$ の間にあれば、 $t>1$ の範囲で2点で交わる。
>     $$-2 < -3a < -1 \iff \mathbf{\frac{1}{3} < a < \frac{2}{3}}$$
> 
> (3) ①が正の解と負の解を1つずつもつ
>     正の解 $\iff t > 1$、 負の解 $\iff 0 < t < 1$ である。
>     $\iff$ 不等式 $g(t) = -3a$ が **$0 < t < 1$ の範囲に1つ、 $t > 1$ の範囲に1つ**の実数解をもつ。
>     グラフより、水平な直線 $Y = -3a$ が、 $t=1$ のときの $y$ 座標 $-1$ から、 $t=0$ のときの $y$ 座標 $2$ の間にあれば、条件を満たす。
>     $$-1 < -3a < 2 \iff \mathbf{-\frac{2}{3} < a < \frac{1}{3}}$$

#### [追加パターンD] 指数を含む連立方程式（基本対称式への帰着）
**【鉄則】指数を含む連立方程式において、 $a^{x+y} = a^x \cdot a^y$ であることに注目し、 $a^x = X, \ a^y = Y$ と置き換えることで、和 $X+Y$ と積 $XY$ の対称な連立方程式（実数解 $t^2 - (X+Y)t + XY = 0$）に帰着させよ！**

> **【例題 (問題集P.146-147 Q351類題)】**
> 次の連立方程式を解け。
> $\begin{cases} 3^x + 3^y = 12 \\ 3^{x+y} = 27 \end{cases}$
> 
> **【思考プロセスと解答】**
> 1. $3^x = X, \ 3^y = Y$ とおく。指数関数の性質より、 **$X > 0, \ Y > 0$** ･･･① である。
> 2. 与えられた連立方程式を $X, Y$ を用いて書き換える。 $3^{x+y} = 3^x \cdot 3^y = XY$ であるから、
>    $\begin{cases} X + Y = 12 \\ XY = 27 \end{cases}$
> 3. $X, Y$ は、 $t$ の2次方程式 $t^2 - 12t + 27 = 0$ の2つの解である。
>    $$(t-3)(t-9) = 0 \implies t = 3, \ 9$$
> 4. ①（ $X > 0, Y > 0$ ）を満たしていることを確認し、 $X, Y$ の組を特定する。
>    $$(X, Y) = (3, 9) \quad\text{または}\quad (X, Y) = (9, 3)$$
> 5. それぞれを $x, y$ に戻して解を求める。
>    *   $(3^x, 3^y) = (3, 9) \implies \mathbf{(x, y) = (1, 2)}$
>    *   $(3^x, 3^y) = (9, 3) \implies \mathbf{(x, y) = (2, 1)}$
> 6. よって、求める解は $\mathbf{(x, y) = (1, 2), \ (2, 1)}$。
